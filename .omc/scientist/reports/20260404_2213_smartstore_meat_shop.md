## PART 1: 시작 전 필수 허가/신고 사항

[FINDING] 정육(소고기) 온라인 판매는 일반 식품과 달리 **3단계 별도 신고**가 필수이며, 이를 누락하면 스마트스토어 계정 정지 및 과태료 대상이 된다.

[STAT:n] 법적 근거: 축산물위생관리법 시행규칙 제35조, 통신판매업신고 의무

### 1-1. 필수 신고 3종 세트

| 단계 | 신고 종류 | 신청 기관 | 소요 기간 | 비용 |
|------|----------|----------|----------|------|
| 1 | **식육판매업 신고** | 시/군/구청 축산물위생과 | 3~7일 | 무료 |
| 2 | **사업자등록** | 세무서 / 홈택스 | 1~3일 | 무료 |
| 3 | **통신판매업 신고** | 시/군/구청 또는 정부24 | 1~3일 | 무료 |

**총 준비 기간**: 7~14일  
**주의**: 닭고기/오리는 일정 조건에서 식육판매업 신고 없이 통신판매 가능하나, **소고기(한우)는 반드시 식육판매업 신고 필요**.

### 1-2. 식육판매업 신고 필요 서류

1. 영업신고서 (축산물 판매업)
2. 사업장 평면도 (냉장고 위치 포함)
3. 냉장/냉동 시설 증빙 (냉장 0~10℃, 냉동 -18℃ 이하 유지 가능한 설비)
4. 건강진단결과서 (보건증) - 지역 보건소 발급, 약 3만원
5. 식품위생교육 수료증 - 온라인 가능, 6~8시간, 2~4만원

### 1-3. 스마트스토어 가입 조건

- **sell.smartstore.naver.com** 접속
- 사업자 등록번호 입력 (개인사업자 가능)
- 통신판매업 신고번호 입력
- 정산 계좌 등록
- 승인: 통상 1~3 영업일

[STAT:n] 스마트스토어 개설 비용: 0원 (입점비/보증금 없음)

## PART 2: 스마트스토어 초기 세팅 (실전 체크리스트)

[FINDING] 초기 세팅의 완성도가 초기 노출 순위를 직접 결정한다. 특히 네이버쇼핑 연동 활성화와 배송 템플릿 설정은 가장 먼저 해야 할 필수 작업이다.

### 2-1. 스토어 기본 설정

```
[체크리스트]
□ 스토어명: 검색 가능하고 기억하기 쉬운 이름 (예: "XX 한우 직배송")
□ 스토어 URL: 영문으로 단순하게
□ 대표 카테고리: 식품 > 수산/축산/정육 > 소고기

□ 출고지 주소 = 사업자등록증 주소
□ 반품/교환지 주소 등록
□ 정산 계좌 등록
□ 네이버쇼핑 연동 활성화 (스토어관리 > 채널연동) ← 필수!
```

### 2-2. 상품 등록 최적화

**상품명 작성 원칙**:
- 길이: 25~30자 이내
- 형식: `[브랜드/산지] 핵심키워드 세부키워드 중량/규격`
- 예시: `냉장 한우 갈비 1kg 가정용 명절 선물 직배송`
- 금지: 동일 단어 3회 이상 반복 (SEO 페널티)
- 금지: 특수문자 과다 사용

**상품 이미지**:
- 대표 이미지: 1000 x 1000px 이상, 고화질
- 실제 고기 사진 (잘 정돈된 트레이/포장 상태)
- 생산지/등급 인증 인증서 이미지 포함 권장
- 동영상 첨부 시 알고리즘 가산점

**상품 상세 페이지 필수 항목** (미기재 시 노출 불이익):
- 축산물 원산지, 등급, 중량, 보관방법, 소비기한
- 영양성분표
- 업체명/소재지 (식품 표시 법적 의무)

**배송 방법**: 택배

**배송비 전략 (정육점 특화)**:

**추천 택배사 (냉장 정육 기준)**:

## PART 3: 하루 2건을 위한 SEO 및 노출 전략

[FINDING] 네이버 쇼핑 SEO는 3개 핵심 요소(적합도, 인기도, 신뢰도)로 구성되며, 초보 셀러는 **적합도(키워드 최적화)**와 **신뢰도(정보 완성도)**에 집중하는 것이 가장 비용효율적이다.

[STAT:effect_size] 리뷰 10개 돌파 시 클릭률 약 2.4배 증가 (네이버 쇼핑 알고리즘 기준)

### 3-1. 알고리즘 3요소 이해

| 요소 | 가중치 | 초보가 통제 가능? | 핵심 액션 |
|------|--------|-----------------|----------|
| 적합도 (Relevance) | 높음 | YES | 상품명, 카테고리, 태그 최적화 |
| 인기도 (Popularity) | 높음 | 부분적 | 클릭률, 찜, 판매건수, 리뷰수 |
| 신뢰도 (Trust) | 중간 | YES | 정보 완성도, 규정 준수 (페널티 방지) |

### 3-2. 정육 특화 키워드 전략

**메인 키워드 (경쟁 과다 → 초기 회피)**:
- "한우 갈비", "한우 목심" → 대형 판매자와 정면 경쟁, 초보에게 불리

**추천 세부 키워드 (롱테일 → 초기 공략)**:
- `냉장 한우 목심 1kg 직배송`
- `가정용 한우 갈비 소포장`
- `한우 갈비 1인분 냉장 당일출고`
- `[지역명] 한우 직판 갈비`
- `명절 한우 갈비 선물세트 1kg`

**키워드 분석 무료 도구**:
1. **네이버 데이터랩** (datalab.naver.com): 검색어 트렌드 확인
2. **네이버 광고 시스템** (manage.searchad.naver.com): 검색량/경쟁도 무료 조회
3. **아이템스카우트** (itemscout.io): 상품 키워드 분석 (무료 플랜 있음)
4. **판다랭크** (pandarank.net): 경쟁자 분석

**상품명 작성 예시 (최적화)**:
```
나쁜 예: "한우 한우갈비 한우목심 갈비 소고기 최고급"
좋은 예: "냉장 한우 갈비 1kg 소포장 직배송 가정용"
```

### 3-3. 카테고리 설정

- 이동 경로: 식품 > 수산/축산/정육 > 소고기 > 갈비/갈비살 (또는 목심)
- 타겟 키워드 검색 후 **1위 상품과 동일 카테고리** 확인 후 맞춤 설정

### 3-4. 태그 및 속성 완성도

```
필수 태그: 한우, 냉장, 갈비, 목심, 1kg, 직배송, 소고기
속성 입력: 원산지(국내산), 등급(1+등급 등), 중량, 보관방법
```

[FINDING] 속성을 빠짐없이 입력하는 것만으로도 노출 순위에 가산점이 부여된다.

## PART 4: 첫 리뷰 10개 확보 전략

[FINDING] 리뷰 0개 상태에서는 네이버쇼핑 상단 노출이 사실상 불가능하다. 첫 10개 리뷰가 가장 중요한 투자 대상이다.

[STAT:n] 리뷰 10개 이하: 상단 노출 제한, 신상품 표시 유지; 리뷰 10개 초과: 클릭률 2.4배 상승

### 4-1. 초기 리뷰 확보 방법 (합법적)

**방법 1: 지인 구매 유도 (가장 빠름, 비용 최소)**
- 가족/친구/단골 고객에게 실제 구매 후 리뷰 요청
- 반드시 **네이버페이로 실구매 후 작성**해야 정식 리뷰로 인정
- 현금 보상보다 상품 할인 쿠폰으로 제공 (약관 위반 리스크 최소화)
- 목표: 2주 내 5~7개 확보

**방법 2: 배송 시 리뷰 요청 카드 동봉**
- 손편지 또는 인쇄 카드: "솔직한 리뷰 남겨주시면 다음 구매 시 XX% 할인"
- QR 코드로 리뷰 작성 페이지 바로 연결
- 텍스트만 있는 리뷰보다 **사진 리뷰가 SEO 가산점 더 높음**

**방법 3: 알림톡 리뷰 유도**
- 배송 완료 후 3~5일째 자동 알림 발송 (네이버 톡톡 활용)
- 문구 예시: "고기 맛있게 드셨나요? 솔직한 리뷰가 큰 힘이 됩니다 :) 리뷰 작성 시 다음 주문 1,000원 할인"

**방법 4: 체험단 (블로그 체험단)**
- 1인당 비용: 상품 원가 + 약 3만원 (체험단 플랫폼 수수료)
- 플랫폼: 아싸뷰(assaview.co.kr), 레뷰, 리뷰파트너
- 주의: 네이버는 스마트스토어 내 리뷰는 **실구매자만 작성 가능** → 블로그 체험단은 블로그 노출용이며 스토어 리뷰와 별개
- 블로그 포스팅이 네이버 블로그 검색에 노출되어 유입 증가 효과

**피해야 할 방법**:
- 리뷰 구매 (마케팅 대행사 허위 리뷰) → 네이버 적발 시 스토어 정지
- 카카오톡 리뷰 부탁 시 현금 지급 → 약관 위반

### 4-2. 리뷰 10개 달성 예산

| 방법 | 비용 | 기간 |
|------|------|------|
| 지인 5명 할인 구매 (5,000원 할인x5) | 25,000원 | 1~2주 |
| 리뷰 요청 카드 인쇄 (100장) | 5,000~10,000원 | 상시 |
| 블로그 체험단 3명 | 약 90,000~150,000원 | 2~4주 |
| **합계** | **약 12~18만원** | **1개월 내** |

## PART 5: 수수료 구조 완전 분석

[FINDING] 2025년 6월 개편으로 기존 "네이버쇼핑 유입 수수료 2%"가 폐지되고, "판매 수수료"로 재편되었다. 셀러가 직접 마케팅 링크로 유입시킬 경우 수수료가 절반 수준으로 줄어든다.

[STAT:n] 샘플 계산: 35,000원 주문 1건 기준

### 5-1. 2025년 6월 이후 수수료 구조

| 수수료 종류 | 요율 | 적용 조건 |
|-----------|------|---------|
| 네이버페이 주문관리 수수료 | 1.98% ~ 3.63% | 모든 주문 (매출 규모별) |
| 판매 수수료 (일반) | 2.73% | 네이버쇼핑 검색/플러스스토어 유입 |
| 판매 수수료 (셀러 마케팅) | 0.91% | 판매자 직접 마케팅 링크 유입 |
| 직접 유입 | 0% | 외부 SNS/블로그 직접 유입 |

**초보 셀러 (연 매출 3억 미만): 네이버페이 수수료 3.63% 적용**

### 5-2. 실전 수익 계산 예시

**시나리오: 한우 갈비 1kg, 판매가 45,000원, 배송비 무료 포함**

```
판매가: 45,000원
- 네이버페이 수수료 (3.63%): -1,634원
- 판매 수수료 (2.73%): -1,229원
- 포장/냉장비: -9,400원 (여름 기준)
- 한우 원가 (1kg, 갈비): 약 25,000~30,000원
─────────────────────────────
실수익: 약 2,740~7,740원 / 건
```

[FINDING] 한우 원가가 높은 구조상, 판매가 설정이 수익의 핵심 변수다. 최소 가격은 배송비+포장비+수수료를 고려해 원가의 1.5~1.8배 이상으로 책정해야 한다.

### 5-3. "스타트 제로수수료" 프로그램

- 대상: 개설 후 1년 미만, 일정 매출 이하 신규 판매자
- 혜택: 네이버페이 수수료 일부 감면
- 신청: 스마트스토어센터 > 정산/수수료 메뉴에서 확인

## PART 6: 초기 마케팅 전략 (예산별)
