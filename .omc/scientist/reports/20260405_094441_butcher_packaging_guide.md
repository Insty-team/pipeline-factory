## 1. 포장 방식 선택: 진공포장 vs 트레이(산소)포장

| 항목 | 진공포장 | 트레이(산소)포장 |
|------|---------|----------------|
| 유통기한 | 냉장 6~10주 | 냉장 10일~2주 |
| 비용 | 비닐봉투 단가 낮음 | 트레이+랩 비용 발생 |
| 외관 | 고기가 눌려 보임 | 고기 형태 그대로 보임 |
| 택배 적합성 | **매우 적합** (밀봉) | 충격에 약함 |
| 설비 | 진공포장기 필요 (3~8만원) | 트레이+랩 기계 필요 |
| 추천 | **온라인 택배에 강력 추천** | 오프라인 진열용 |

**결론: 온라인 택배 판매에는 진공포장이 압도적으로 유리합니다.**  
진공포장은 배송 중 핏물 유출이 없고, 유통기한이 길어 반품 리스크가 낮습니다.

## 2. 진공포장기 추천 제품 및 가격

### 입문용 (가성비 우선, 3~5만원대)
| 제품명 | 가격 | 특징 |
|--------|------|------|
| 보만 음식 밀봉기 | 약 2.4~3만원 | 터치 한 번으로 작동, 롤비닐 포함 |
| 에펠 EF-SU813 | 약 4.2만원 | 50dB 저소음, 원터치 작동 |
| 해피락 더존 | 약 5만원 | 가성비 우수, 업소용 전환 가능 |
| 위즈웰 WH6010 | 약 3~4만원 | 보존기간 5배 향상 |

**구매처:** 쿠팡, G마켓, 이마트몰(SSG), 다나와 비교 후 구매

### 사용 팁 (노부부 친화적)
- 버튼 1~2개짜리 **원터치 제품** 선택
- 롤비닐 포함 제품 선택 (별도 구매 번거로움 없음)
- 소음 50dB 이하 제품 권장

### 진공포장 비닐봉투 소모품 구매처
- **팩솔 (packsol.kr)**: 업소용 진공비닐 다양한 사이즈
- **포장백서 (packing100.kr)**: 두께 0.05~0.10mm 선택 가능
- **우림테이프 (woolimtape.com)**: 가정용·업소용 모두 판매
- **다산팩 (dasanpack.com)**: 공장직영 최저가

## 3. 택배 포장 재료 구매처 (도매·대량)

### 스티로폼 박스 (보냉 박스)
| 업체 | 특징 | 연락/주소 |
|------|------|----------|
| 경기포장몰 (1pack.co.kr) | 보냉팩·스티로폼·포장자재 전문 | 온라인 주문 가능 |
| 경원스티로폼 (kwbox.co.kr) | 국내제조 150여종, 대량주문 견적할인 | 서울경기 당일배송 |
| 태성산업 (ts8.co.kr) | 아이스팩+스티로폼 세트, 대량구매 무료배송 | 온라인 주문 가능 |

### 아이스팩 보관 온도 기준
> **핵심 공식:** 물 100% 아이스팩 2개 + 적절한 크기의 스티로폼 박스  
> = 축산물 1kg 기준 48시간 동안 10°C 이하 유지 가능

| 스티로폼 박스 크기 | 아이스팩 수량 | 권장 육류 무게 |
|-----------------|-------------|--------------|
| 소형 (2~3리터) | 아이스팩 2개 | 500g~1kg |
| 중형 (5~7리터) | 아이스팩 3~4개 | 1~2kg |
| 대형 (10리터+) | 아이스팩 5~6개 | 2~3kg |

### 소량 주문 가능한 곳
- 쿠팡·G마켓에서 "스티로폼 아이스박스 50개입" 검색
- 보마내팩 (bomnae-pack.com): 소량도 주문 가능
- 우림테이프 (woolimtape.com): 낱개~대량 모두 가능

## 4. 단계별 포장 순서 (노부부 실전 매뉴얼)

```
[STEP 1] 고기 손질 및 소분
  - 주문 단위(500g, 1kg 등)로 저울에 달아 소분
  - 키친타올로 핏물 가볍게 제거
  - 냉장 상태 유지 (상온 노출 최소화)

[STEP 2] 진공포장
  - 진공포장 비닐봉투에 고기 투입
  - 봉투 입구 기포 없이 넣기
  - 진공포장기 작동 → 자동 밀봉
  - 포장 후 냉장·냉동 보관

[STEP 3] 라벨 부착
  - 제품명, 중량, 원산지(국내산 한우), 포장일자
  - 유통기한, 보관방법(냉장 0~4°C 또는 냉동) 기재
  - 스티커 크기: 가로 3cm × 세로 2cm 이상 (법적 최소)

[STEP 4] 스티로폼 박스 준비
  - 박스 바닥에 아이스팩 1개 배치
  - 진공포장된 고기 넣기
  - 고기 위에 아이스팩 1~2개 추가
  - 빈 공간은 뽁뽁이(에어캡)로 채우기 (흔들림 방지)

[STEP 5] 외부 포장
  - 스티로폼 뚜껑 닫고 테이프로 X자 봉인
  - 스티로폼 박스를 골판지 박스에 넣기 (선택사항)
  - 외부에 "냉장/냉동 보관" 스티커 부착
  - 운송장 출력 후 부착

[STEP 6] 택배 접수
  - 냉장: 쿨택배 서비스 이용 (CJ대한통운, 로젠택배 등)
  - 냉동: 냉동택배 서비스 이용
  - 우체국 택배도 냉장 서비스 가능 (농촌 지역 편리)
```

| 항목 | 예시 | 비고 |
|------|------|------|
| 제품명 | 한우 등심 | |
| 식육 종류 | 쇠고기 (한우) | |
| 원산지 | **국내산 한우** | 반드시 "한우" 명시 |
| 중량 | 500g | 저울로 실측 |
| 포장일자 | 2026.04.04 | |
| 유통기한 | 냉장: 포장 후 5일, 냉동: 3개월 | |
| 보관방법 | 냉장(0~4°C) 또는 냉동(-18°C 이하) | |
| 판매자명 및 주소 | ○○정육점, 전라북도 ○○군 | |
| 연락처 | 010-XXXX-XXXX | |

### 글자 크기 규정
- 최소 **12포인트** 이상
- 스티커 최소 크기: **가로 3cm × 세로 2cm** 이상
- 바탕색과 다른 색으로 선명하게 표시

### 법적 근거
- 식품등의 표시기준 (식품의약품안전처)
- 농수산물의 원산지 표시요령 (국립농산물품질관리원)

## 6. 브랜드 스티커 제작 방법

### 저비용 옵션 (소량 주문)
| 업체 | 최소 주문 | 특징 | 가격대 |
|------|---------|------|--------|
| 인새랑 (insaerang.co.kr) | 소량 가능 | 감성 라벨, 코팅 선택 | 장당 50~100원대 |
| 레드프린팅 (redprinting.co.kr) | 소량 가능 | 자유형 스티커 | 소량도 가능 |
| 아이라벨 (label.kr) | 1장도 가능 | PDF/JPG 업로드 | 소량 높은 단가 |
| 롤라벨 (printingkorea.net) | 롤 단위 | 바코드 포함 가능 | 대량 저렴 |

### 초간단 DIY 스티커 (비용 최소화)
1. 네이버 무료 도구 "CLOVA Note" 또는 **미리캔버스(miricanvas.com)** 에서 무료 라벨 디자인
2. A4 라벨지(사무용품점 구매, 장당 300~500원) 구매
3. 가정용 프린터로 출력 후 붙이기
4. **비용: 장당 약 10~20원**

### 스티커 필수 정보 포함 체크리스트
- [ ] 가게 이름 (상호)
- [ ] 고기 종류 및 부위
- [ ] 원산지: 국내산 한우
- [ ] 중량
- [ ] 포장일자 / 유통기한
- [ ] 연락처

## 7. 고기 사진 촬영 팁 (스마트폰)

### 핵심 원칙 3가지
1. **자연광 활용**: 창가에서 낮 시간대 촬영. 형광등 X, 햇빛 O
2. **배경 단순화**: 흰 접시, 나무 도마, 한지(한지 느낌 천) 위에 고기 올리기
3. **위에서 아래로**: 45도 각도 또는 정면에서 촬영 (옆면 강조)

### 단계별 촬영 방법
```
1. 스마트폰 카메라 앱 열기
2. 화면의 고기 부분을 손가락으로 탭 (초점 고정)
3. 화면 밝기 조절 (햇빛 너무 강하면 약간 어둡게)
4. 수평/수직 격자선 켜기 (카메라 설정에서 '격자' 활성화)
5. 3등분선 교차점에 고기 중심 오도록 구도 잡기
6. 사진 3~5장 연속 촬영 후 가장 좋은 것 선택
7. 기본 사진 앱에서 밝기/채도 약간 올리기
```

### 고기 사진을 맛있게 보이게 하는 꿀팁
- 고기 표면의 **마블링(결)이 보이도록** 자르기
- 촬영 전 **키친타올로 물기 제거** (윤기 있어 보임)
- 한우라면 **참기름 한 방울** 살짝 바르면 윤기 남
- 배경에 마늘, 파, 된장 등 부재료 곁들이면 한국적 느낌

## 8. 비용 절감 전략 요약

### 초기 투자비용 (최소 구성)
| 항목 | 제품 | 예상 비용 |
|------|------|---------|
| 진공포장기 | 보만 또는 에펠 원터치 | 3~5만원 |
| 진공포장 비닐 (100매) | 팩솔, 우림테이프 | 1~2만원 |
| 스티로폼 박스 (20개) | 경기포장몰, 태성산업 | 2~4만원 |
| 아이스팩 (20개) | 태성산업, 쿠팡 | 1~2만원 |
| 라벨 스티커 (100매) | A4 라벨지 + 프린터 | 5천~1만원 |
| **합계** | | **약 8~14만원** |

### 운영 단계 절감 방법
1. **소재를 구매할 때는 묶음 구매**: 비닐봉투·아이스팩 50~100개 단위 구매 시 30~50% 저렴
2. **스티로폼 박스 재사용 불가** → 대신 소형 사이즈 정확히 맞춰 쓰기 (큰 박스 = 아이스팩 더 필요)
3. **아이스팩 재냉동 재사용 가능**: 받은 고객이 반납할 필요 없음, 자체 보유분 재사용
4. **라벨 직접 출력**: A4 라벨지 + 프린터 = 외주 대비 90% 절감
5. **택배사 계약**: 월 50건 이상이면 택배사 직접 계약 시 건당 500~1,000원 절감

### 택배비 절감
- **우체국 EMS 농산물 우대 요금** 확인 (농촌 지역 할인 있음)
- **CJ대한통운 스마트픽** 등 온라인 계약으로 10~15% 할인
- 택배 묶음 발송 (화·목 정기 발송) → 픽업 기사 스케줄 관리
