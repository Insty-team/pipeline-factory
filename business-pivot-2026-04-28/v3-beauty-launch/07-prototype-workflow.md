### 3-1. 컨셉 정하기 (10분)

5장 흐름:

- 3장: **디자인 1** + 짧은 설명
- 4장: **디자인 2** + 짧은 설명
- 5장: **CTA** (예약 방법, "사장님께 DM 또는 카톡")

`03-cardnews/concept.md`에 5장 텍스트 미리 적어둠.

### 3-2. 미리캔버스 작업 (60분)

1. https://www.miricanvas.com 접속, 가입
2. **새로 만들기** → **카드뉴스 (1080×1080)**
3. **템플릿** 검색: "네일", "뷰티", "심플" 등 — 마음에 드는 템플릿 1개
4. 색상 변경 — `brand-tone.md`의 색 팔레트로 통일
5. 폰트: 한국어 잘 보이는 1세트 (제목/본문)
6. **5장 채우기** — 시술사진 + 짧은 텍스트
7. 다 만든 후 **다운로드** → PNG 5장 또는 PDF 1개
8. 저장: `03-cardnews/cardnews-v1-1.png ~ cardnews-v1-5.png`

⚠️ **함정 회피**: 무료 템플릿 그대로 쓰면 다른 가게와 동질화. **색·폰트·1요소 최소 1번 바꾸기**.

## Phase 4 — 릴스 1편 (90분)

### 4-1. 사진 보정 (15분)

6~10장 골라 일괄 보정.

방법 A (Lightroom Mobile):
- 1장 보정 (밝기 +5~10, 채도 -2~5, 대비 살짝)
- 그 설정을 preset으로 저장
- 나머지 사진에 일괄 적용

방법 B (Snapseed, 무료):
- 1장 보정 → 우상단 ⋮ → "이미지 조정 복사" → 다른 사진 열어서 "이미지 조정 붙여넣기"

저장: `04-reels/photos-edited/`

⚠️ **과도 보정 금지** — 시술 결과 왜곡 = 컴플레인 트리거.

### 4-2. CapCut 영상화 (30분)

1. CapCut 열기 → **새 프로젝트** (9:16 세로, 1080×1920)
2. **템플릿** 탭 → "네일", "뷰티" 검색 → 1개 골라 **사용**
3. 보정한 사진 6~10장 끼우기
4. 자동 트랜지션 적용
5. **내보내기** → `04-reels/draft-v1.mp4`

⚠️ 첫 시도면 30초 길이 부담 → **15초**로 시작해도 OK. 다음 시제품에 30초.

### 4-3. Vrew 자막 (15분)

1. Vrew 열기 → 새 프로젝트 → CapCut export한 mp4 import
2. 자막 자동 생성 (음성 없으면 텍스트 직접 입력)
3. 자막 폰트/색 가게 톤에 맞춤
4. 내보내기

⚠️ 자막은 **짧고 굵게**. 30초에 자막 4~6개면 충분.

### 4-4. 음악 + 마무리 (30분)

1. CapCut에 자막 입힌 영상 다시 import (또는 Vrew에서 직접 음원 추가)
2. **음악** 탭 → 트렌딩 음원 (네일/뷰티 분위기) — 인스타 트렌드 음원 우선 (저작권 안전)
3. 30초 길이로 컷, 끝나는 부분 fade out
4. 최종 내보내기: `04-reels/final.mp4`
