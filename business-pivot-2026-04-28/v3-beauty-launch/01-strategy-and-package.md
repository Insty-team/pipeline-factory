# 01. 전략 + 패키지 정의

## 1. 핵심 전략 (3줄)

1. **시장 95% 카피**: 크몽 베스트셀러(아이엔in 스탠다드 52.8만, 거래 197건) 패키지 구조 그대로
2. **5%만 차별화**: DM/카톡 자동응답 — 1인샵 페인 1순위(시술 중 응대 불가), 크몽 10건 중 0건이 다룸
3. **Sell-before-build**: 5곳 무료 시제품 → 응답 2~3곳 → 4주 무료 베타 → 유료 전환 / 지인 소개

## 2. 첫 패키지 정의

### "1인 네일·에스테틱 인스타 + AI 응대 자동화 패키지"

**유료 가격: 월 59만원** (베타 4주는 무료)

**포함 항목:**
- 월 10회 인스타 업로드 (이미지 8 + 시술 카드뉴스 2)
- 시술사진 AI 자동 보정 → 릴스 1편/주
- **DM·카톡 자동 응대 봇 1회 셋업 + 월간 학습** ⭐ 차별화
- 월간 성과 보고 (팔로워·DM 전환·예약 건수)

**점주 입력:** 시술사진만 카톡으로 전송. 끝.

### 가격 근거

- 크몽 베이직 44~88만 / 스탠다드 70~120만 (네일/뷰티 SNS)
- 49만은 신뢰 부족, 99만은 1인샵 부담 → **59만이 sweet spot**
- 챗봇 셋업(시장 60만 가치)을 6.2만 마진으로 끼워넣음 = 같은 가격에 응대 페인 해결

## 3. 시제품 vs 베타 구분 (중요)

| 단계 | 길이 | 무엇 | 목적 |
|---|---|---|---|
| **시제품** | 1회성 (몇 시간 분량) | 캡션 5개 + 카드뉴스 1세트 + 릴스 1편 + 챗봇 데모 | DM 첫 인사용 — "이거 만들어봤어요, 보실래요?" |
| **무료 베타** | 4주 운영 | 패키지 풀 운영 (월 10회 업로드 + 챗봇 + 보고) | 효용성 입증 → 유료 전환 |

→ **시제품 5곳 발송**은 OK (각 4~6시간). **베타는 2~3곳이 현실 한계** (5곳 베타 = 주 12시간 초과).

## 4. 무료 → 유료 전환 시나리오

### Week 0: 시제품 발송
- 5곳 시제품 만들어 DM
- 응답 2~3곳 기대 (40~60% 응답률 목표)

### Week 1: 베타 시작 (응답 온 1~3곳)
- "한 달 무료로 운영해드릴게요. 대신 첫 케이스라 결과 측정하고 후기 1줄만 부탁드려요."
- **베타 시작 시 명시적으로 동의 받기** (문자/카톡 캡처):
  - 측정할 지표 동의 (KPI 4개, 아래 5번 참고)
  - 후기 1줄 약속
  - 베타 종료 시 유료 전환 또는 종료 결정 약속

### Week 1~4: 베타 운영
- 매주 진척 짧게 공유 (월요일 카톡: "이번주 게시물 N건 도달 X명, DM Y건 자동응답")
- 매주 점주 피드백 1줄 받기 ("이건 좋았어요" / "이건 아쉬워요")

### Week 4 마지막날: 유료 전환 제안
- 결과 데이터 정리 → "이렇게 나왔어요" 1장 보고서
- 다음 메시지 그대로:

> 4주 운영 결과 정리해드릴게요.
>
> - 팔로워: [이전] → [이후] (+X명, +Y%)
> - 게시물 평균 도달: [이전] → [이후]
> - DM 자동응답: 총 X건 처리 → 추정 놓친 예약 Y건 회수
> - [그 외 가게별 의미 있는 지표]
>
> 두 가지 중에 편하신 거 골라주시면 돼요.
> 1) 계속 돌리고 싶으시면 다음 달부터 월 59만으로 운영해드릴게요.
> 2) 부담스러우시면, 이런 거 필요하실 만한 다른 사장님 한 분만 소개해주셔도 충분해요.
>
> 어느 쪽이든 괜찮고, 둘 다 안 되셔도 4주 운영해드린 결과물은 그냥 가져가세요.

### 전환률 가정
- 베타 1곳 → 유료 전환 30~40% / 소개 40~50% / 둘 다 X 20~30%
- 베타 5건 사이클 = 유료 1.5~2건 + 소개 2~3건 (소개로 다음 사이클 가속)

## 5. 효용성 측정 KPI (베타 시작 시 점주와 합의)

| 지표 | 측정 방법 | 베타 목표 |
|---|---|---|
| 팔로워 증가 | 인스타 인사이트 | +5~15% (4주) |
| 평균 도달/노출 | 인스타 인사이트 | +20~30% |
| **DM 응답률·속도** ⭐ | 챗봇 로그 | 24/7 1차 응답, 평균 5분 이내 |
| **놓친 예약 회수** ⭐ | DM에서 예약 문의 → 실제 예약 전환 | 정성 추적 (점주 인터뷰) |
| 후기 1줄 | 베타 종료 시 받음 | 필수 |

→ ⭐ 두 개가 핵심. 다른 셀러가 못 보여주는 데이터.

## 6. 본인 신뢰 자산 (왜 너에게 맡길까)

DM에 한 줄로 끼워 넣을 본인 소개. 다음 중 1~2개:

- "AI/자동화 도구 만드는 일 하고 있어요" (대중 모름 — 약함)
- "최근에 의료기관 마케팅 데이터 다루다가, 1인샵에 더 큰 빈자리 보여서 시작했어요" (구체)
- "지금 사장님이 첫 번째 베타라, 저도 진짜 신경 써서 만들 거예요" (정직 — 약점이 강점)

→ **권장: "첫 베타라 정직하게 신경 쓴다"** 톤. 거짓 포장 X. 메모리 `feedback_promo_tone` 일관.

## 7. 백업 시나리오

| 상황 | 대응 |
|---|---|
| 5곳 발송 → 0건 응답 | DM 메시지 재검토. 다음 5곳 새 후보 발굴해서 재발송. 4주 내 베타 1건 확보 목표 |
| 1건 응답했지만 베타 거절 | 시제품만 무료 제공, 추후 재컨택. 다음 5곳 발송 |
| 베타 운영 중 점주 이탈 | 회고 (왜 이탈?) → KPI 정의 / 패키지 / DM 톤 점검 |
| 베타 4주 결과 미흡 | 결과 솔직하게 공유, 무리한 유료 제안 X. 다음 사이클로 이동 |
