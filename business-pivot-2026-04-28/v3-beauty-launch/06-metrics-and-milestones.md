# 06. KPI + 마일스톤 (200만 → 1,000만)

## 1. 우리(나)의 KPI — 매주 측정

### 영업 funnel

| 단계 | 측정 | 목표 (M1~M3) | 목표 (M4~M6) |
|---|---|---|---|
| 시제품 발송 | 누적 건수 | 5~10건/월 | 5~10건/월 |
| 발송 → 본 | % | 60%+ | 70%+ |
| 본 → 응답 | % | 30%+ | 40%+ |
| 응답 → 시제품 미팅 | % | 60%+ | 70%+ |
| 미팅 → 베타 시작 | % | 50%+ | 60%+ |
| 베타 → 유료 전환 | % | (M3 첫 측정) | 30~40% |
| 베타 → 지인 소개 | % | (M3 첫 측정) | 40~50% |

### 운영 효율

| 지표 | 목표 |
|---|---|
| 시제품 1세트 제작 시간 | M1 5h → M3 3h → M6 2h (자동화 효과) |
| 베타 1곳 주간 운영 시간 | M1 3h → M3 2h → M6 1.5h |
| **본인 한계 클라이언트 수** | M1 1 / M3 3 / M6 5 / M12 8~10 / M24 12~15 |

## 2. 클라이언트(점주)의 KPI — 베타 효용성 입증

### 모든 베타 시작 시 baseline 캡처 + 4주 후 비교

| 지표 | 측정 방법 | 최소 목표 | Stretch 목표 |
|---|---|---|---|
| **팔로워 증가** | 인스타 인사이트 | +5% (4주) | +15% |
| **평균 도달/노출** | 인스타 인사이트 | +20% | +40% |
| **DM 자동응답 처리** ⭐ | 챗봇 로그 | 누적 30건+ (4주) | 100건+ |
| **DM 1차 응답 시간** | 챗봇 로그 | 평균 5분 이내 | 1분 이내 |
| **놓친 예약 회수** ⭐ | 점주 인터뷰 + DM 추적 | 정성 추적 | 1~3건 정량 회수 사례 |
| **점주 만족도** | 베타 종료 시 1줄 후기 | "도움 됨" 평가 | "계속 쓰고 싶음" |

### 베타 시작 시 점주와 합의할 1장 (구두 + 카톡 캡처)

```
4주 무료 베타 합의서 (간단 버전)

1. 기간: YYYY-MM-DD ~ YYYY-MM-DD (4주)
2. 무료 운영 항목: 인스타 10회 업로드 + 릴스 4편 + DM 자동응답
3. 측정 지표 (양쪽 동의):
   - 팔로워, 도달, DM 처리 건수, 응답 시간
4. 점주 협조: 시술사진 주 5~10장 카톡 전송, 베타 종료 시 후기 1줄
5. 베타 종료 시점:
   - 효용성 입증되면: 월 59만 유료 전환 또는 지인 1명 소개 (택 1)
   - 효용성 미흡하거나 부담스러우면: 자유 종료, 결과물은 그냥 가져가셔도 됨
```

## 3. 단계별 마일스톤 (현실 기준)

### M1 (2026-05) — Day 1~30

**목표**: 시제품 5곳 발송, 베타 1~2곳 시작

| 주 | 액션 | 성공 기준 |
|---|---|---|
| W1 | 도구 셋업 + 시제품 1곳 (@nail_beige) | 발송 1건 |
| W2 | 시제품 2~3곳 추가 발송 | 누적 발송 3건, 응답 1건+ |
| W3 | 응답 1~2건 → 시제품 미팅 / 베타 제안 | 베타 0~1건 시작 |
| W4 | 시제품 4~5곳 완료, 베타 1~2건 운영 시작 | 베타 진행 1~2건 |

**M1 손익**: 매출 0, 비용 약 5만 (Canva Pro + ChatGPT)

### M2 (2026-06) — Day 31~60

**목표**: 베타 1차 운영 + 유료 전환 첫 시도

| 주 | 액션 | 성공 기준 |
|---|---|---|
| W5~6 | 베타 W2~3 운영 + 신규 시제품 2~3건 | 신규 응답 2건+ |
| W7 | 베타 W4 + 결과 정리 + 유료 전환 제안 | 첫 유료 전환 시도 |
| W8 | 유료 전환 결과 + 다음 베타 사이클 | 유료 0~1건 / 소개 1~2건 |

**M2 손익**: 매출 0~50만, 비용 5만 = -5만 ~ +45만

### M3 (2026-07) — Day 61~90

**목표**: **첫 매출 발생**

| 주 | 액션 | 성공 기준 |
|---|---|---|
| W9~12 | 유료 1건 안정 운영 + 베타 2~3건 신규 + 지인 소개로 유입 | 월말 유료 1건(50만) + 베타 진행 2~3건 |

**M3 손익**: **매출 50만**, 비용 5만, 순이익 약 45만 (시간당 약 6,000원 — 아직 낮음)

### M6 (2026-10) — Day 150 → 손익분기 통과

**목표**: **월 매출 200만**

- 유료 클라이언트 4~5건 × 49~59만 = 200~295만
- 베타→유료 전환률 35%, 6사이클 누적 가정
- 패키지 안정화: 현재 패키지 1개 → 베이직 39만 / 스탠다드 59만 / 프리미엄 99만 도입

**M6 손익**: 매출 200~250만, 비용 약 10만 (도구 + 카톡 챗봇 유료 플랜 검토), 순이익 약 190~240만

### M12 (2027-04) — Day 360

**목표**: **월 매출 400만**

- 유료 7~8건 × 평균 55만 = 385~440만
- 프리미엄 99만 클라이언트 1~2건 진입
- AI 자동화 파이프라인 V1 (캡션·카드뉴스 자동 생성)
- 본인 시간 30~40% 자동화 절감 → 클라이언트 수 늘릴 여유

**M12 손익**: 매출 400만, 비용 30만 (도구 + 자동화 인프라), 순이익 약 370만

### M18 (2027-10)

**목표**: **월 매출 600~700만**

- 유료 10~12건
- 외주 작가 1명 파트타임 (월 100~150만 비용)
- 본인은 영업 + 자동화 + 클라이언트 관계만

### M24 (2028-04)

**목표**: **월 매출 1,000만**

- 유료 12~15건 + 프리미엄 3~5건
- 외주 1~2명 또는 AI 자동화 V2 (영상 자동, 인스타 자동 발행)
- 단가 평균 65~70만 (프리미엄 비중 ↑)
- **본인 시간 = 영업/자동화/관계, 콘텐츠 제작 X**

**M24 손익**: 매출 1,000만, 비용 약 200~300만 (외주 + 도구 + 인프라), 순이익 약 700~800만

## 4. 1,000만 도달의 진짜 enabler

| Enabler | 우선순위 | 시점 |
|---|---|---|
| **AI 자동화 V1** (캡션·카드뉴스 자동) | 1 | M3~M6 |
| **AI 자동화 V2** (영상·자동 발행) | 2 | M9~M12 |
| **외주 1명** (파트타임 작가) | 3 | M12~M18 |
| **프리미엄 패키지 99만** (의료법 가드레일 강화) | 4 | M6~M9 |
| **소개 채널 시스템화** (지인 소개에 인센티브) | 5 | M3~M6 |
| **본인 채널** (인스타·블로그 — "1인샵 마케팅 비서") | 6 | M6~M12 |

→ 클라이언트 수만 늘리는 식의 1,000만은 1인 한계로 X. **자동화 + 외주 + 프리미엄 단가 ↑** 3축이 같이 가야 함.
