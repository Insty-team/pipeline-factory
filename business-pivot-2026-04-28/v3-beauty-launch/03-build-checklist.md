# 03. 빌드 체크리스트 (시제품 + 도구)

## 1. 도구 셋업 (총 1시간, 시제품 단계는 거의 무료)

> **중요 업데이트 (2026-04-30 시장조사 반영):** 한국 1인샵 표준은 미리캔버스(Canva 아님) + CapCut/VLLO(둘 다). Canva Pro는 베타 시작 후 검토.

| 도구 | 용도 | 학습 | 비용 | 우선순위 |
|---|---|---|---|---|
| **미리캔버스** ⭐ | 카드뉴스 (한국 1인샵 표준) | 1~2시간 | 무료 | A |
| **CapCut** | 릴스 템플릿 편집 (글로벌 표준) | 4시간 (유튜브 강의 1편) | 무료 | A |
| **VLLO** ⭐ | 모바일 영상 편집 (한국 1위) | 1시간 | 무료 (워터마크) | A |
| **Vrew** | 한국어 자막 자동 + 음성 변환 | 2시간 | 무료(워터마크) → 월 9천 | A |
| **ChatGPT Plus / Claude** | 캡션·답글·해시태그 생성 | 0 (이미 사용 중) | 월 2~3만 | A |
| Canva Pro | 카드뉴스 (영문 템플릿 더 풍부) | 2시간 | 월 14,900원 | C (베타 시작 시 검토) |
| **캐럿(Carat)** | AI 카드뉴스 자동 생성 (한국어) | 30분 | 무료 (2025.5 확대) | B (베타 시작 시 시도) |
| **RoboMD** | Meta 공식 파트너, AI 피드 자동 | 1시간 | 유료 | C (M3+ 검토) |
| Notion | 응답 추적 + 클라이언트 관리 | 0 | 무료 | B |
| **채널톡** (또는 카카오 챗봇) | DM 자동응답 | 1~2시간 | 무료 플랜 충분 | A (베타 시작 시) |
| Manychat | 인스타 DM 자동응답 | 1~2시간 | 무료~월 1.5만 | B (베타 시작 시) |

→ A 등급만 먼저. B는 베타 1건 확보 후. 시제품 단계는 무료 도구로 충분.

## 2. 시제품 1세트 구성 (1곳당 4~6시간)

### 인풋
- 그 가게 인스타 최근 게시물 시술사진 5~10장 (캡처)
- 가게 톤/위치/특징 (관찰)

### 아웃풋

| # | 결과물 | 도구 | 소요 |
|---|---|---|---|
| 1 | **AI 캡션 5개** (시술별, 인스타 톤 + 해시태그 30개) | ChatGPT/Claude | 30분 |
| 2 | **카드뉴스 1세트** (3장: 표지+소개+CTA) | Canva | 1~1.5시간 |
| 3 | **릴스 1편** (30초, 시술사진 6~10장 + Vrew 자막 + 트렌드 음악) | CapCut + Vrew | 1.5~2시간 |
| 4 | **DM 자동응답 데모** (3가지 시나리오 답변) | ChatGPT 프롬프트 또는 채널톡 | 30분~1시간 |
| 5 | **간단 보고서 1장** (이렇게 만들었어요 PDF) | Canva 또는 Notion | 30분 |

### DM 자동응답 데모 시나리오 (3가지 필수)

```
시나리오 1: "예약 가능한가요?"
→ "안녕하세요! 사장님 시술 중이실 수 있어 자동으로 답변드려요. 
   네일베이지 예약은 [예약 링크 또는 카톡 채널]에서 가능하시고,
   궁금하신 디자인 사진 보내주시면 사장님이 시술 끝나는 대로 직접 안내드릴게요."

시나리오 2: "[디자인] 가능해요? 가격은요?"
→ "네 그 디자인 충분히 가능해요! 정확한 가격은 손톱 길이/연장 여부에 따라 달라서, 
   카톡으로 손톱 사진 한 번 보내주시면 사장님이 직접 견적 드릴게요. 
   참고로 비슷한 디자인은 평소 [범위] 정도예요."

시나리오 3: "오늘/내일 가능해요?"
→ "당일·익일 예약은 사장님 일정 직접 확인이 필요해서, 
   카톡 [채널 링크]로 한 번 메시지 주시면 가장 빠르게 답변드릴 수 있어요."
```

→ 점주 톤에 맞게 매번 살짝 커스터마이즈. ChatGPT 프롬프트 한 번 짜두고 재사용.

## 6. 자동화 우선순위 (시간 절약 = 사업 핵심)

베타 1~2건 안정되면, 다음 자동화 투자:

| 자동화 | 예상 시간 절감 | 난이도 | 우선순위 |
|---|---|---|---|
| 캡션 생성 프롬프트 + 자동화 (n8n/Make) | 주 30분 | ★★ | 1 |
| 시술사진 → 카드뉴스 템플릿 (Canva 자동화) | 주 1시간 | ★★★ | 2 |
| 릴스 자동 자막 + 자동 게시 예약 | 주 30분 | ★★★ | 3 |
| DM 분류·답변 학습 강화 | 주 1시간 | ★★★ | 2 |
| 주간 보고서 자동 생성 | 주 30분 | ★★ | 4 |

→ M3~M6에 자동화 강화하면 **1인 한계 5건 → 10~15건 가능**. 이게 M24 1,000만의 진짜 enabler.
