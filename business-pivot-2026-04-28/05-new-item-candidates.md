## 2. Top 3 상세 평가

### ⭐ Top 1 — **C1. 1인 학원·공부방 카톡/AI 챗봇 자동응답 도입 대행**

| 기준 | 적용 | 통과 |
|---|---|---|
| 1. 노동 베이스 | OpenAI API 키만 있으면 됨. 카톡채널 무료. 첫 1년 비용 < 100만원 | ✅ |
| 2. 시장 우선 + 작음 | 전국 학원 약 8만 개 중 **1인 운영 공부방·소형 학원**(전체 미용업 X 패턴 동일). 1인이 100~300곳 영업 가능 | ✅ |
| 3. 하락 X | 학원 시장은 5년 flat~소폭 성장. AI 챗봇 도입은 S4(크몽 9개월 1.2만 의뢰), S14(도입 효과 검증)로 **가파른 성장 보너스** | ✅✅ |
| 4. 검증된 어휘 + 빈자리 | "카톡 자동응답" / "챗봇 도입"은 검증된 어휘. **학원 vertical에 특화한 챗봇 대행은 크몽 10건 중 0건**(범용 챗봇 개발은 많음) → 빈자리 명확 | ✅✅ |
| 5. 작은 문제 + 빠른 시제품 | 학원 페인: "야간/주말 학부모 문의 응대 불가" → 챗봇 1세트(자주 묻는 Q 30개 학습) **4~6h 안에 시제품 1개** 가능 | ✅ |
| 6. 통제권 + 관심 | 직접 영업(원장 카페/네이버 학원 검색 → DM·전화). AI 도구·프롬프트 자산 누적. 사용자가 **AI/자동화에 강한 관심** | ✅ |

**점수: 6/6 (즉시 진행 가능)**

**핵심 영업 한 줄**: "학원 원장님 야간 학부모 문의, 카톡 챗봇이 자동으로 답합니다. 4시간이면 세팅."
**차별화 한 줄**: "범용 챗봇이 아닌 **학원 vertical 전용 프롬프트 + 학부모 자주 묻는 30 Q&A 학습 키트** 포함."

**리스크/보강**:
- 학원 원장은 신기술에 보수적 → **무료 1주 베타 1건**으로 신뢰 확보 후 유료화 (sell-before-build 원칙).
- "학원" → 1인 공부방 / 소수정예 영어학원 / 코딩학원 등 sub-vertical로 더 좁히는 게 유리.

### ⭐ Top 2 — **C2. 1인 동네샵 숏폼 릴스 제작·운영 대행 (사장 출연형)**

| 기준 | 적용 | 통과 |
|---|---|---|

| 2. 시장 우선 + 작음 | 전국 1인 카페/공방/플라워샵/베이커리 vertical. 1인이 의미 있게 점유 가능 | ✅ |
| 3. 하락 X | **숏폼 시장 CAGR 60%**(S2), 크몽 거래량 **9배**(S1). 가파른 성장 ✅✅ | ✅✅ |
| 4. 검증된 어휘 + 빈자리 | "릴스 대행"은 검증 어휘(크몽에 200개+ 거래). **빈자리: "사장 본인 출연 + 대행이 촬영·편집·업로드까지 풀패키지"** — 대부분은 편집만 또는 모델 별도. 1인샵 vertical로 좁히면 0건에 가까움 | ⚠️→✅ |
| 5. 작은 문제 + 빠른 시제품 | 페인: "혼자라 촬영·편집·업로드까지 못 함" → 샘플 릴스 1편 4~6h 가능 (현장 1h 촬영 + 3~5h 편집·업로드) | ✅ |
| 6. 통제권 + 관심 | 직접 DM·동네 방문 영업. 자체 IP(편집 템플릿) 자산화. 사용자 영상 콘텐츠 관심도 高 | ✅ |

**점수: 5.5/6 (진행 추천 — 빈자리는 vertical 좁힘으로 보강)**

**핵심 영업 한 줄**: "사장님이 1시간만 시간 내주시면, 한 달치 릴스 12편을 저희가 다 합니다."
**차별화 한 줄**: "편집만 하는 곳 X. **현장 촬영 + 편집 + 업로드 + 댓글 응대까지 풀패키지**, 1인 카페 전용 톤·자막 템플릿."

**리스크/보강**:
- 경쟁이 가장 치열함 (크몽 검색 결과 다수). → vertical 매우 좁게(예: "1인 베이커리 only") 잡고 포트폴리오로 차별화.
- 시장 폭발 vs 단가 하락 동시 진행(S1: 80%가 2만원 미만) → **월 정기권** 모델로 단가 방어 필요.

4. **7일 후 결과 측정**: 학부모 응답 자동화 비율 → 유료 전환 여부.
