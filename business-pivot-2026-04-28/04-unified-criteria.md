# 04. 통합 아이템 선정 기준 (2026-05-02 작성)

> 두 영상(5기준 영상 + 방법 1번 영상) + 사용자 원래 직관을 합쳐서 만든 우리만의 6기준.
> 새 아이템·새 vertical을 검토할 때마다 이 표를 채워서 **5/6 이상이면 진행, 미달이면 다음 후보로 넘김**.

## 출처

- `01-criteria-review.md` — 5기준 영상 (#1 노동 / #2 작은 시장 / #3 성장 / #4 익숙 / #5 통제권)
- 방법 1번 영상 — 시장 우선의 법칙 / 표준 없는 시장 / 작고 구체적 문제 / 빠른 출시
- 사용자 원래 직관 — 현재 팔리고 있는 / 내 관심 / 노동집약적 / 충분히 작은 시장

## 충돌 해결 메모 (적용 시 참고)

| 충돌 | 해결 방식 |
|---|---|
| 5기준 #4 "익숙한 아이템" vs 방법 1 "표준 없는 시장" | **두 층으로 분리.** 고객에게는 익숙한 어휘(영업 wrapper), 공급에는 빈자리(차별화 angle). v3가 이미 이 패턴 — 95% 카피 + 5% DM 자동화. |
| 5기준 #3 "하락 X" (방어) vs 방법 1 "가파른 성장" (공격) | **둘 다 둠.** 최소 통과 = 하락 아님 / 보너스 = 가파른 성장. |
| 빠른 출시 vs sell-before-build | **같은 의미.** 우리에게 "기본 기능"은 "수동 deliverable / 베타 1건". 코드 X. |

## 6기준

| # | 기준 | 최소 통과 | 보너스 | 테스트 (yes/no) |
|---|------|----------|--------|-----------------|
| **1** | **노동 베이스** | 자본·재고·고정비 ≤ 월 10만 | 0원 | 첫 1년 누적 비용 < 200만인가? |
| **2** | **시장 우선 + 작음** | 시장을 먼저 정함 + 1인이 의미 있게 점유 가능 | 큰 시장 안의 좁은 vertical (예: 1인샵 only) | 한 줄로 "누구의 어떤 시장"인지 즉답 가능한가? |
| **3** | **하락 X (가파른 성장이면 +)** | 네이버 트렌드 5년 flat 이상 + 매장수/사업자수 안 줄어듦 | CAGR + / 인접 카테고리 확장 중 / 신규 매장 증가 | 네이버 트렌드 + 통계청 매장수 5년 그래프 캡처해봤는가? |
| **4** | **검증된 어휘 + 빈자리 angle** | 고객이 30초 안에 "아 그거"라고 이해 (크몽/숨고에 비슷한 거래 존재) + 같은 시장에 표준화 안 된 작은 angle 1개 | 빈자리가 "크몽 10건 중 0건" 수준으로 명확 | 영업 한 줄 + 차별화 한 줄을 따로따로 즉답 가능한가? |
| **5** | **작고 구체적 문제 + 빠른 deliverable** | 고객이 자기 입으로 말한 페인 + 4~6시간 안에 만들 수 있는 시제품 단위 | 시제품 1개로 즉시 가치 시연 가능 | "오늘 4~6시간이면 이거 1세트 만든다"가 가능한가? |
| **6** | **통제권 + 내 관심** | 직접 영업·자체 채널 + 6개월 이상 흥미 유지 가능한 분야 | 자체 도메인·도구 자산 누적 가능 | 플랫폼 차단되면 망하는 구조인가? (X면 통과) + 6개월 후에도 이 시장 글 읽고 싶을 것 같은가? |

### 통과 룰

- **6/6**: 즉시 진행
- **5/6**: 진행하되, 미달 항목은 4주 안에 보강하거나 재검토
- **4/6 이하**: 다음 후보로 넘어감 (메시지 재포장으로 해결될 수준 아님)

### 보강 vs 재검토 기준

- **메시지/wrapper로 보강 가능**: #4 (어휘 재포장), #2 (vertical 재정의)
- **데이터로만 보강 가능**: #3 (트렌드 캡처)
- **본질적 미달 = 재검토**: #1 (자본 필요), #5 (4시간 안에 시제품 X), #6 (플랫폼 의존, 관심 X)
