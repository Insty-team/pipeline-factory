# 케이스 스터디: 흙수저 무자본 창업 6단계 (월 1억까지)

> 출처: 유튜브 '돈린이' 영상
> 핵심: **무자본 → 월 1억** 6단계 로드맵 + 실행 마인드셋

---

## 무자본 창업 6단계

| 단계 | 액션 | 기간 |
|---|---|---|
| **1. 마인드셋** | 부자들 책 읽기, 사고방식 간접 경험 | 큰 방향 잡기 |
| **2. 필수 스킬** | 글쓰기, 마케팅, 심리학 (책+강의) | 공통 역량 |
| **3. 전문 지식 1개 마스터** | 디자인/홈페이지/마케팅 중 택1, **1개월 집중 학습** | 실무 역량 |
| **4. 플랫폼 + 무료 마케팅** | 크몽/숨고 등록 + 커뮤니티 활동 | **수익 X, 후기 ✓** |
| **5. 가격 인상 + 레버리지** | 실무는 프리랜서, 본인은 마케팅 집중 | **월 1천만 가능** |
| **6. 직원 채용** | 무리한 확장 X, 프리랜서 비중 조절 | 안정 운영 |

---

## 핵심 철학 — 끌어당김의 법칙 비판

> "끌어당김 X, **'존나 열심히 하는 태도'** O"
> 종교적 안식처 의존 X, **매일 목표 + 계획 + 실행** ✓

---

## 우리 사업 진단 — 단계별 자가 평가 ⭐

| 단계 | 평가 | 상태 |
|---|---|---|
| 1. 마인드셋 | 6개 케이스 영상 + 메모리 4개 정리 완료 | ✅ |
| 2. 필수 스킬 | 마케팅(케이스 2,3) + 카피(케이스 1,3) + 심리(케이스 6) 학습 완료 | ✅ |
| 3. 전문 지식 1개 | **마케팅 + AI 자동화** 결합 (의료법 가드레일) | 🔄 1주차 |
| 4. **플랫폼 + 무료 마케팅** | 크몽 등록 + 카카오 오픈톡 (케이스 1) | ⏳ **다음 단계** |
| 5. 가격 인상 + 레버리지 | 베타 3건 → 49만 → 79만 → 109만 | ⏳ 3개월 후 |
| 6. 직원 채용 | 1년 후 단계 | ⏳ 12개월+ |

→ **우리는 정확히 4단계 직전.** 1~3단계 다 갖춰짐. 남은 건 **플랫폼 등록 + 무료 마케팅 시작**.

### 4단계 실행 체크리스트
- [ ] 크몽 셀러 등록 (사업자/세금계산서 발행 가능)
- [ ] 카카오 오픈채팅방 5곳 가입 (피부관리/네일/필라테스 사장님 모임)
- [ ] 첫 무료 가치 콘텐츠 1편 ("사장님이 모르는 AI 마케팅 5가지")
- [ ] 베타 3건 영업 시작

## 7개 케이스 통합 — 영상이 모두 같은 결론으로 수렴

| 케이스 | 우리 단계와의 매핑 |
|---|---|
| 1. 잘팔렙 | 4단계 무료 마케팅 (카카오 오픈톡) + 5단계 업셀 |
| 2. 마케팅 본질 | 2단계 필수 스킬 |
| 3. 5역량 | 2단계 필수 스킬 + 5단계 시스템화 |
| 4. 6기준 | 1단계 마인드셋 (검증) |
| 5. 본질 (만족) | 1단계 + 4단계 (만족 → 후기) |
| 6. 심리 (상호성/일관성) | 4단계 영업 클로징 |
| 7. 6단계 (본 영상) | **전체 로드맵 = 우리 위치 = 4단계 직전** |

## 핵심 한 줄

**"끌어당기지 말고 실행하라. 우리는 이미 4단계 직전이다."**

7개 영상이 1~6단계 다 검증해줌. 더 이상 학습/분석 단계 아님. **이번 주 안에 4단계 시작이 정답.**

→ **Action Item 1순위:** 크몽 셀러 등록 + 카카오 오픈톡 5곳 리스트업 + 베타 1명 DM.
