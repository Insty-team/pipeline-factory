# 케이스 스터디: 사람 심리 조종 2가지 법칙 (상호성 + 일관성)

> 출처: 유튜브 영상 요약 (24살 월 5천만+ 크리에이터)
> 핵심: **도덕적 논란 가능, 정직하게 쓰면 강력한 비즈니스 무기**

---

## 1. 상호성의 원칙 (Reciprocity)

**원리:** 인간은 호의를 받으면 보답해야 한다는 **부채의식** 느낌. 원시 시대 생존 본능.

### 핵심 전략
**결제 유도 전 → 무상 호의 먼저 → 심리적 부채 심기**

### 강력한 사례
- 디자인 회사: 결제 전 **30분 무료 컨설팅** → **90%+ 의뢰 전환**
- 마트 시식 코너, 화장품 샘플 = 같은 원리

---

## 2. 일관성의 법칙 (Consistency)

**원리:** 인간은 자기 말/행동을 지키려는 경향. 정서적 에너지 절약 = "입장 정립".

### 핵심 전략
**상담 중 고객 스스로 "신뢰 간다"고 말하게 유도** → 그 말에 책임 → 결제로 이어짐

### 강력한 사례
- 상담 끝에 "왜 저희 회사를 선택하셨나요?" 질문
  → 고객의 긍정 답변 자체가 자기 입장 정립
  → 자연스럽게 의뢰 클로징
- 장난감 회사 크리스마스 품절 → 부모의 "약속" → 재입고 시 무조건 구매

---

## 결론

**도덕성이 핵심**. 거짓말 배제 + 정직하게 활용 → 강력한 비즈니스 무기.

---

## 우리 사업 적용 — 영업 스크립트에 즉시 반영 ⭐

### 상호성 원칙 적용 (이미 부분 적용 중)
| 단계 | 호의 (무상 제공) | 의도 |
|---|---|---|
| 1차 접촉 | **무료 의료법 가드레일 진단** (30분) | 부채의식 + 전문성 노출 |
| 2차 미팅 | **샘플 카드뉴스 1장** (사장님 시술 사진으로) | 결과물 미리보기 = 부채 ↑ |
| 3차 클로징 | **베타 3건은 첫 달 환불 보장** | 부채 + 안전망 |

→ 영상 사례 (90% 전환율) 우리에게 그대로 적용 가능. 베타 3건이 곧 상호성 패키지.

### 일관성 법칙 적용 — 영업 클로징 스크립트
**미팅 마지막 질문 시퀀스:**

```
1. "원장님 가게가 다른 곳과 다른 강점이 뭐라고 생각하세요?"
   → 사장님이 자기 가게 강점 말함 (긍정 입장 정립)

2. "그 강점이 환자한테 잘 전달되고 있다고 느끼세요?"
   → "잘 안 되는 것 같다" 답변 유도 (필요 자각)

3. "오늘 들으신 것 중에 어떤 부분이 가장 와닿으셨어요?"
   → 사장님 입으로 우리 가치 말하게 (자기 입장 정립)

4. "그럼 그 부분 같이 풀어볼까요?"
   → 자연스러운 클로징 (자기 말 지키려는 심리)
```

→ 영상 사례를 사장님 미팅에 그대로 이식.

## 6개 케이스 통합 — 영업 메시지 빌딩 블록

| 케이스 | 영업에 쓸 핵심 한 줄 |
|---|---|
| 1. 잘팔렙 | "다른 업체 견적 한번 받아보세요. 비교하면 답 나옵니다." (대조) |
| 2. 마케팅 본질 | "원장님, 광고 본 고객이 다음에 뭐 할 것 같으세요?" (전환 자각) |
| 3. 5역량 | "시술 중에 못 답하시잖아요. 그 시간에 신환 5명 놓치십니다." (필요) |
| 4. 6기준 | (메시지 X, 우리 자신 검증) |
| 5. 본질 | "30일 안에 신환 +5건 또는 100% 환불입니다." (만족 보장) |
| 6. 심리 (본 영상) | "오늘 어떤 부분이 가장 와닿으셨어요?" (일관성 클로징) ⭐ |

### 통합 영업 흐름 (10분 시나리오)
```
[1분] 인사 + 라포
[2분] 페인 자각 — "광고 본 고객 다음 행동" 질문 (케이스 2)
[2분] 필요 만들기 — "시술 중 응답 못함 = 신환 5명 손실" (케이스 3)
[2분] 가치 제시 — "주 6h + 월 200만 절약" (케이스 3)
[1분] 대조 — "200~300만 대행 vs 우리 49만" (케이스 1)
[1분] 안전망 — "30일 환불 보장" (케이스 5)
[1분] 일관성 클로징 — "어떤 부분이 와닿으셨어요?" (케이스 6)
```

## 핵심 한 줄

**"호의를 먼저 베풀고, 사장님이 직접 말하게 만들어라."**

상호성 + 일관성 = 구매 확률 90%+. 5번 케이스(만족 본질)가 도덕적 베이스를 깔아주고, 6번이 그 위에서 결제를 클로징하는 도구.
