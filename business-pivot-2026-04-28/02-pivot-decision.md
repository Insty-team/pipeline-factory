# 피벗 옵션 + 추천 아이템

## 피벗 옵션 3가지

### Option A: GEO 폐기, 익숙한 서비스 처음부터
- 예: 네이버 플레이스 단독, 블로그 마케팅, 홈페이지 제작
- 장점: 5기준 깔끔히 통과, 즉시 영업 가능
- 단점: **기존 GEO 자산(geo-seo-claude, 13곳 감사, 메시지 v1~v3.1, 발송 4곳 데이터) 전부 폐기.** 차별화 0, 크몽 레드오션에 그대로 들어감.

### Option B: GEO 그대로, 메시지만 재포장
- 예: "AI 시대 SEO" "ChatGPT 마케팅"으로 어휘 변경
- 장점: 자산 유지
- 단점: 여전히 익숙하지 않음. "AI 시대"도 추상적. #4 실패 그대로.

### Option C: 익숙한 서비스로 영업 → GEO를 안에 끼워서 전달 ★ 추천
- 영업 wrapper: "의료/뷰티 클리닉 디지털 점검 + 네이버 플레이스"
- Deliverable: 네이버 플레이스 정비 + 홈페이지 SEO/Schema + AI 검색 시연 + 의료광고법 점검
- 장점: 5기준 모두 통과, GEO 자산을 차별화 무기로 살림
- 단점: 패키지 구성·가격 재설계 필요 (1~2주 작업)

→ **선택: Option C**

## 추천 아이템: 의료/뷰티 클리닉 디지털 점검 패키지 (가칭)

### 한 줄 정의

의료/뷰티 클리닉 사장님이 익숙한 "홈페이지 + 네이버 플레이스" 어휘로 영업하고, 안에 GEO/Schema/AI 검색 진단까지 묶어서 차별화하는 패키지.

### 패키지 구성

| 단계 | 가격 | 내용 | 예상 시간 |
|------|------|------|-----------|
| **무료 진단** | 0원 | 네이버 플레이스 점수 + 홈페이지 SEO 점검 + AI 검색 결과 캡처 PDF | 1~2시간 |
| **셋업** (1회) | 30~50만원 | 플레이스 정보 정비 + 홈페이지 Schema/메타 + robots.txt + llms.txt + 의료광고법 1차 점검 | 4~6시간 |
| **월정액** | 10~20만원 | 월간 점수 리포트 + AI 검색 결과 모니터링 + 분기 업데이트 | 1~2시간/월 |

### 차별점 (다른 크몽 업체 vs 우리)

- 진단 PDF에 **"ChatGPT/Claude에게 물어봤을 때 결과"** 캡처 포함 → 사장님이 처음 보는 시연 = 강력
- 의료광고법 점검 자동화 (기존 GEO 도구 응용)
- 홈페이지 Schema/구조화 데이터 → 다른 업체는 안 함

### 5기준 매핑

| # | 기준 | 통과 여부 | 근거 |
|---|------|-----------|------|
| 1 | 노동 베이스 | ✅ | 서비스업, 재고 0 |
| 2 | 작은 시장 | ✅ | 의료/뷰티 한정 (전체 클리닉이 아닌) |
| 3 | 성장 시장 | ✅ | 네이버 플레이스 + AI 검색 둘 다 성장 (Week 1 검증) |
| 4 | 특별하지 않은 아이템 | ✅ | **네이버 플레이스 = 모든 클리닉이 이미 함** |
| 5 | 통제권 | ✅ | 직접 영업, 자체 도구·도메인 |

### 200~300만/월 도달 경로

- 셋업 30만 × 7건/월 = 210만
- 또는 셋업 30만 × 4건 + 월정액 15만 × 8건 = 120 + 120 = 240만
- 6~12개월에 걸쳐 retainer 누적 → 안정

### 영업 메시지 v4 헤드라인 후보

- "[클리닉명] 네이버 플레이스 점수 무료 진단 보내드려도 될까요?"
- "[클리닉명] 홈페이지가 ChatGPT 검색에 어떻게 나오는지 보여드릴 수 있을까요?"
- "[클리닉명] 디지털 노출 무료 점검 — 5분 PDF 1장"
