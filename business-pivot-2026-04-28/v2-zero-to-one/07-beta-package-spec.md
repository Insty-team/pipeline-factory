# 07. 베타 패키지 사양 + 사장님 1명 확보 준비

> 2026-04-30 결정 사항 정리. 내일부터 이어서 실행.

---

## 베타 정책

- **1명 먼저** (네일샵 1곳)
- **검증 성공 시 → 3명까지 확장**
- **무료** (시장가 환산 월 39~49만 가치)
- **기간:** 30일

---

## 풀패키지 사양 (30일)

| 항목 | 내용 |
|---|---|
| 진단 | 인스타 + 네이버플레이스 + 경쟁 3곳 비교 |
| 콘텐츠 | 카드뉴스 8장 + 릴스 4편 |
| DM 자동화 | 분류 봇 셋업 + 운영 |
| 후기 답글 | 자동 생성 (인스타 + 네이버) |
| 단골 리텐션 | 카톡 시퀀스 셋업 |
| 리포트 | 매주 KPI 추적 (1회/주, 총 4회) |

---

## 교환 조건 (사장님 → 우리)

- 후기 1개
- 만족 시 친구 사장님 1명 소개 (상호성 원칙)
- 미달 시 부담 없이 종료

→ 사장님 입장에서 **압도적 유리한 거래**로 보여야 미팅 거절 안 나옴.

---

## 사장님 후보 결정 가이드 — 가장 먼저 정해야 할 것

| 옵션 | 장점 | 단점 |
|---|---|---|
| 친구/지인 | 미팅 즉시 잡힘, 솔직 피드백 | 객관성 부족 |
| **친구의 친구** ⭐ | 거리감 적당 + 객관성 | 약간의 영업 필요 |
| 콜드 컨택 (인스타 DM) | 케이스 가치 최대 | 답장률 낮음 |

→ **추천: 친구의 친구.** 신뢰 시그널 있고 객관성도 적당.

---

## 1단계: 미팅 전 (30분~1시간)

| 준비물 | 시간 | 담당 |
|---|---|---|
| ① 사장님 후보 1명 결정 | 15분 | **너** |
| ② 인스타 URL + 네이버플레이스 URL | 5분 | **너** |
| ③ 미팅 약속 잡기 (대면/줌 60분) | 10분 | **너** |
| ④ 무료 패키지 안내서 PDF 1장 | 24h 내 | Claude |
| ⑤ 권한 위임 동의서 1장 | 24h 내 | Claude |

---

## 2단계: 미팅 중 (60분)

미팅에서 받아 올 6가지:

| # | 항목 | 받는 방법 |
|---|---|---|
| 1 | 시술 사진/영상 원본 50장+ | 핸드폰 에어드롭/카톡 |
| 2 | 지난 DM 샘플 30개 | 인스타 DM 스크린샷 |
| 3 | 사장님이 쓴 답글 10개 | 플레이스/인스타 댓글 캡처 |
| 4 | 단골 DB (이름+전화) | 카톡 채널 또는 예약장부 |
| 5 | 사장님 페인 1~3개 | 그냥 들으면 됨 |
| 6 | 계정 권한 위임 동의 | 1장짜리 동의서 |

---

## 3단계: 미팅 후 1주차 (셋업)

**Claude/시스템 작업:**
- DM 자동 분류 봇 셋업
- 답글 톤 학습 + 자동 생성기
- 단골 리텐션 카톡 시퀀스
- 카드뉴스 4장 (1주차 분량)
- 진단 리포트 1차

**너 작업:**
- 카드뉴스 게시 (사장님 인스타 계정)
- 사장님과 카톡 응대 (주 30분 평균)

---

## 시간 산정 (주 12~14h 제약 검증)

| 주차 | 시간 | 비고 |
|---|---|---|
| 1주 (셋업) | 8~10h | 미팅 1h + 셋업 보조 + 톤 가이드 |
| 2~4주 (운영) | 2~3h/주 | 게시 + 카톡 응대 |
| **합계 (30일)** | **15~20h** | 주 평균 5h |

→ 1주차만 빡빡. 2주차부터 여유.

---

## 사장님 안내서 — 핵심 카피 (미리보기)

> **"30일 동안, 사장님이 평소에 6시간씩 쓰시던 콘텐츠/DM/답글 작업을 0시간으로 줄여드립니다."**
>
> ✓ 시술 사진 → 카드뉴스 8장 + 릴스 4편 (인스타 그대로 사용 가능)
> ✓ 시술 중 DM 자동 분류 (예약/문의/광고)
> ✓ 후기 답글 자동 생성
> ✓ 단골 자동 카톡 (3주 후 자동발송)
> ✓ 매주 진행 리포트
>
> **조건:** 30일 끝에 만족하시면 후기 1개 + 친구 사장님 1명 소개.
> 미달이면 그냥 안 하시면 됩니다.

---

## "연습"의 진짜 가치 (재정의)

이번 1명 베타 = 단순 마케팅 패키지가 아님:
1. **시스템 자동화 PoC** — 어디까지 AI로 처리되는지 검증
2. **소요 시간 측정** — 4번째 사장님부터 진짜 단가 책정 가능
3. **케이스 라이브러리** — 다음 영업에서 "강남 ○○네일 사례" 자산

→ **베타 1건 = 우리 사업의 PoC + 시간 견적 + 영업 자료** 한 번에 확보.
