# 05 — 유어라인 사장님 사용 시나리오 시뮬레이션

> 사장님이 시제품을 받은 뒤 **하루·일주일·4주** 동안 어떤 화면을 보고, 어떤 카톡을 받고, 어떤 결정을 하게 되는지 시뮬레이션. 사장 미팅·DM·교육 자료에 그대로 활용 가능.
> 작성: 2026-05-03

---

## 0. 한 줄 요약

> **사장님이 시술 사진 1장만 카톡 보내면, 디지털 매니저 1명이 4주 동안 무료로 인스타·블로그·후기·예약·단골 관리·매출 측정까지 다 해주고, 사장님은 폰으로 대시보드 1탭만 열면 매일 매출이 얼마 늘었는지 볼 수 있는 시스템.**

---

## 1. 사장님이 받는 4가지 채널 (DM 발송 직후)

| # | 채널 | 형식 | 사장 행동 |
|---|---|---|---|
| 1 | **카카오톡 일반 메시지** | 사진 1장 카톡 → AI 자동 게시 결과 미리보기 → "OK" 누르면 게시 | 사진만 보냄 |
| 2 | **카카오톡 1줄 알림** | 매일 8시 (사용자 직접 발송) + **대시보드 링크 포함** | 출근하며 읽고, 궁금하면 1탭 |
| 3 | **대시보드 웹 (모바일 즐겨찾기)** | https://uareline-dashboard.streamlit.app (예시) | 시술 사이 폰으로 슥 봄 |
| 4 | **카카오톡 주간 리포트** | 매주 월요일 5줄 텍스트 | 1분 읽고 1줄 피드백 |

→ 사장은 가입·설치·로그인 1번도 안 함. 카톡 1개 + 즐겨찾기 1탭이 전부.

## 2. 사장님 첫날 (베타 D-Day) — 미팅 직후 30분

| 시각 | 사용자 | 사장 |
|---|---|---|
| 14:00 | 미팅에서 4가지 전달 + 시연 | 시제품 시연 봄 |
| 14:25 | 카톡 채널에서 "사진 1장만 보내주시면 됩니다" 1줄 보냄 | "응 알겠어요" |
| 14:30 | 대시보드 URL 카톡 + 모바일 즐겨찾기 가이드 | 폰에서 즐겨찾기 추가 (30초) |
| 14:35 | "내일 아침 8시부터 매일 1줄 보내드려요" | "ㅎㅎ 좋아요" |
| 14:40 | KPI 4개 + 후기 1줄 약속 + 옵션 4개 카톡 캡처로 보존 | "동의해요" |
| 익일 08:00 | 첫 일간 1줄 발송 | 사장 수신 |

## 3. 사장님 1주일 시뮬레이션 (1주차 기준)

### 월요일 (영업일)

```
07:55  사용자: [유어라인 일일] 어제 신규 후기 +0, 노쇼 0,
       챗봇 문의 1건. 좋은 한 주 시작하세요~♡
       👉 자세히: https://uareline-dashboard.streamlit.app
08:00  사장: 출근하며 1줄 읽음 → 궁금하면 링크 1탭
10:00  사장: 첫 손님 시술 시작 (글루연장 맥스, 60분)
11:30  사장: 시술 종료, 사진 1장 카톡 → 사용자
11:31  AI: 인스타 캡션 5종 + 블로그 초안 사장 카톡으로 미리보기
11:33  사장: "OK" 1번 누름
11:34  AI: 인스타·블로그 동시 게시 + 워터마크 자동 추가
14:00  사장: 시술 사이 폰으로 대시보드 1번 봄
       → "오늘 예약 4건, 챗봇 문의 1건 (예약 전환 1)"
20:00  사장: 영업 종료
       → 내일 예약자 3명에게 자동 알림톡 발송됨 (사장은 모름)
```

### 화요일~목요일
- 매일 비슷한 패턴 (사진 1~2장 카톡, 대시보드 1번 슥 봄)
- 수요일 새 후기 1건 들어옴
  → AI가 사장 톤으로 답글 미리보기 카톡
  → 사장 "OK" → 게시 (사장이 안 본 답글이 자동으로 안 달리게 안전장치)

### 금요일
- 시술 종료 후 대시보드:
  - 이번 주 신규 6명 (전주 +2)
  - 후기 +2건
  - 챗봇 4건 → 예약 2건
  - **4주 베타 누적 매출 기여: +12만원**

### 토·일요일
- 휴무인 날도 일간 1줄은 받음 ("어제 휴무, 챗봇 문의 1건 자동 응답 완료")

## 4. 사장님이 보는 대시보드 — 1페이지 구성

```
🏪 유어라인 대시보드                  마지막 갱신: 17:32

🔴 오늘 (5/15 목)
─────────────────────────────────
예약       4건 (당일 1건)
노쇼       0건
신규 손님  1명
챗봇 문의  2건  →  예약 전환 1
새 후기    0건

📊 이번 주 (5/12 ~ 5/18)
─────────────────────────────────
신규 손님       6명  (전주 +2 ⬆)
신규 후기       3건  (전주 +1 ⬆)
인스타 도달    +18%
알림톡 발송    14건 (예약확정 4·1일전 5·당일 3·재방문 2)

🚨 이탈 위험 단골 (4주+ 미방문)
─────────────────────────────────
김OO  5주차  ✉️ 안부 메시지 발송됨
이OO  6주차  ⏳ 알림 대기 (사장 검수 필요)

💰 4주 베타 누적 매출 기여
─────────────────────────────────
노쇼 회수            22만
재방문 활성화         15만
챗봇 신규 예약        18만
신규 후기 → 신규 손님   8만
단골 회복             10만
─────────────────────────────────
합계                  73만
예상 commission      ~18만 (M5+M6+M9+M4 합산)
```

→ **사장이 commission 가치를 의심할 여지 X** — 항상 보임.

## 5. 사장 의사결정 시나리오 3개

### 시나리오 A — 메디핑크 안 팔림 (W8 약점 대응 진행 중)

```
1주차: 메디핑크 카드뉴스 인스타 게시 (사용자 자동)
2주차: 후기에 메디핑크 키워드 0건  
       대시보드 "메디핑크 신규 시술" = 0
사장 인지: "이 캠페인 효과가 미미하네"
사장 카톡: "사용자님, 메디핑크 가격 더 강조해주세요"
사용자 반응: 다음 주 카드뉴스 가격 강조 버전 + 인스타 캡션 변경
3주차: 메디핑크 시술 +2건 → 대시보드에 표시
4주차: +5건 누적 → 결과 리포트 B에 메디핑크 매출 기여 명시
```

→ **사장이 시제품을 자기 도구로 활용해서 신메뉴 인지 캠페인 직접 운영하는 경험.** 베타 종료 후에도 이 운영 권한을 본인이 갖고 싶어함 = commission 합의 동기.

### 시나리오 B — 단골 이탈 위험 자동 알림 + 회복

```
2주차 화요일:
  대시보드 "이탈 위험 단골" 섹션에 김OO 5주차 미방문 빨간 표시
  → AI가 사장 톤으로 안부 메시지 미리 작성:
    "김OO님~ 한동안 안 보이셔서 ㅎㅎ
     영양제 잘 챙기고 계세요?
     다음 시술 일정 잡으시면 미리 알려드려요♡"
  → 사장 카톡 미리보기 → "OK"
  → 사장 본인 인스타 DM으로 자동 발송
2주차 수요일:
  김OO 답장: "다음 주 화요일 예약할게요!"
  → 단골 회복 1명 = 대시보드 "단골 회복" +1, 매출 기여 +10만
4주차 결과 리포트 B:
  단골 회복 3명 → commission 30~60만 산정
```

### 시나리오 C — 노쇼 줄음 (M5 알림톡 효과 입증)

```
1주차: 노쇼 1건 (알림톡 효과 약함)
2주차: 노쇼 0건
3주차: 노쇼 0건
4주차: 노쇼 1건 (당일 알림 받았는데도 못 옴)

전월 동기: 4주에 노쇼 4건 평균
이번 4주: 노쇼 2건 → -50%
회수 매출 추정: 12만 (시술비 6만 × 2건)

결과 리포트 B:
  노쇼 회수 매출 12만 × commission 25% = 3만
사장 결정:
  "이 정도면 정기 운영도 괜찮네" → 다음 달부터 commission 합의
```

## 6. 사장이 시제품 도중 "이번 주는 못 도와요" 할 때 (안전장치)

> "사장님 이번 주는 시술이 너무 많아서 사진 못 보낼 것 같아요"

| 모듈 | 사장 비협조 시 동작 |
|---|---|
| M1+M2 (인스타·블로그) | 멈춤 (사진 input 없음) — 단, 기존 게시물·해시태그는 유지 |
| M4 (챗봇) | **자동 계속** (사장 input X) |
| M5 (알림톡) | **자동 계속** |
| M6 (후기 답글) | **자동 계속** (사장 검수만 빠짐 → 안전 모드: 답글 보류 → 사용자가 검수) |
| M7 (카드뉴스) | 멈춤 (이미 만든 것만 게시) |
| M9 (이탈 단골) | **자동 계속** |
| M10a~e (대시보드·리포트) | **자동 계속** |

→ 사장 부담 0. **자동화 영역만으로도 commission 가치 일부 유지** = 사장이 "쉬어도 가치 받음" 인지.

## 7. 사장이 4주 후 직접 운영하고 싶을 때 (옵션 4번 — "교육")

```
4주 종료 미팅:
사장: "commission 매월 내는 건 부담이고, 내가 직접 운영하고 싶어요"
사용자: "OK. 교육 옵션 드릴게요"

→ 사용자가 사장에게 인계:
  1. 운영 매뉴얼 PDF (10페이지)
  2. AI 프롬프트 4종 (사장 톤 학습 완료된 것)
  3. 대시보드 비번·소스 코드 (Streamlit Cloud 사장 계정으로 이전)
  4. Google Sheet 권한 이전
  5. 1~2시간 화상 교육 1회
  
사장 자율 운영 시작 (commission X)
사용자 결과:
  - 후기 1줄 ✅
  - 운영 노하우 검증 ✅
  - 다음 사장 소개 받음 ✅ (꼬모래쉬 또는 다른 후보)
  - "유어라인 사례" 마케팅 자산 확보 ✅
```

→ **commission 거절도 출구가 명확**. 사용자에겐 "후기 + 소개 + 사례" 자산 확보, 사장에겐 "자율 운영" 만족.

## 8. 4주 종료 후 사장 결정 path (Decision Tree)

```
4주 결과 리포트 B 받음
     │
     ├─ 매출 기여 명확 (>50만/4주)
     │   ├─ 사장: "정기 운영 부탁드려요"
     │   │  → commission 협상 (예: 측정 매출의 20%)
     │   │  → M10d 월간 PDF C 자동 발송 시작
     │   │  → 다른 사장 소개 1건 자연 발생
     │   │
     │   └─ 사장: "내가 직접 운영하고 싶어요"
     │      → 옵션 4 (교육) 선택 → §7 시나리오
     │
     ├─ 매출 기여 모호 (10~50만)
     │   └─ 사장: "후기는 남길게요, commission은 좀 더 보고"
     │      → 추가 4주 베타 (사장 부담 0)
     │      → 데이터 더 쌓이면 협상 재시도
     │
     └─ 매출 기여 미미 (<10만)
         └─ 사장: "후기 1줄만 남길게요"
            → 거절 정중 수용 → 다음 후보(꼬모래쉬) 진행
            → 학습 자산 확보 (어떤 모듈이 약했는지)
```

## 9. 사장이 받는 지표 — 어디서·언제·어떻게 확인하는지 표

| 지표 | 어디서 | 주기 | 사장 행동 |
|---|---|---|---|
| 오늘 예약·노쇼·신규·챗봇·후기 | 대시보드 (M10a) | 실시간 | 시술 사이 폰으로 슥 |
| 어제 요약 | 카톡 1줄 (M10b) | 매일 8시 | 출근하며 읽음 |
| 이번 주 합계 + 4주 누적 매출 기여 | 대시보드 | 실시간 | 같음 |
| 이탈 위험 단골 | 대시보드 | 실시간 | 알림 메시지 검수 |
| 인스타·블로그 게시 미리보기 | 카톡 (M1+M2) | 사진 보낼 때마다 | "OK" 1번 |
| 후기 답글 미리보기 | 카톡 (M6) | 새 후기 들어올 때마다 | "OK" 1번 |
| 주간 합계 + 트렌드 | 카톡 5줄 (M10c) | 매주 월 | 1분 읽고 1줄 피드백 |
| 4주 결과 + commission 산정 | PDF 리포트 B | 4주 종료 1번 | 미팅에서 함께 봄 |
| 정기 운영 시작 후 월간 합계 | PDF 리포트 C (M10d) | 매월 말 | 보관 |

## 10. 사장이 "왜 이 시제품이 좋은가" 자문자답할 때 떠올리게 될 답

> 1. **시간**: 콘텐츠 작업 주 4~6h → 30분 (월 16~20h 절약)
> 2. **돈**: 노쇼 회수 + 신규 후기 + 챗봇 신규 + 단골 회복 = **월 +30~80만 추정**
> 3. **자산**: 사장 톤 학습된 시스템 — 베타 끝나도 본인 운영 가능 (옵션 4)
> 4. **데이터**: 매일·매주·월간 자동 리포트 — 다음 달 의사결정 근거
> 5. **마음 부담 ↓**: 후기 답글·재방문·문의 자동 → 시술 집중
> 6. **주도권**: 매일 폰으로 대시보드 1탭 → 시스템이 자기 도구라는 감각
