|---|---|---|---|

| **P2** | **노쇼 평균 20%** (5명 중 1명) | 주 1~2건 | 한국경제 / 토스플레이스 |

→ P1·P2·P5·P6은 v3와 공통, **P3·P4는 속눈썹 특이 페인**.

## 3. 이미 시장에 있는 검증된 솔루션 (단품)

| 페인 | 솔루션 | 시장가 | 검증 |
|---|---|---|---|
| P1 | 카카오 채널 + GPT 자동응답 셋업 | 30~60만 1회 셋업 | 크몽 1.2만 의뢰 |
| P2 | 알림톡 자동발송 (예약확정/1일전/당일) | 월 3~10만 (마메드네·공비서) | 도입 후 노쇼 ↓ |
| P3 | (현재 표준 솔루션 없음) | — | **빈자리 angle** |
| P4 | 인스타 운영 대행 (월 8~10건 + 릴스 2~4편) | 월 50~150만 (크몽 표준) | 크몽 200건+ 거래 |
| P5 | 4주 재방문 알림톡 + 쿠폰 자동 발송 | 월 5~10만 (P2 도구 활용) | 단골 방문률 +30% (뷰티인프라 사례) |
| P6 | 네이버 플레이스 운영 + 영수증 후기 유도 | 월 20~40만 | 크몽 표준 |

→ **P3 (디자인 컨설팅) 빼고는 모두 검증된 솔루션 존재**. P3은 우리가 풀패키지에 끼워 넣으면 자연스러운 보너스 (디자인 카드뉴스 자동 제작 = 인스타 콘텐츠와 결합).

→ **빈자리는 솔루션이 아니라 묶음**: 사장이 5~6개 SaaS·대행을 따로 가입·운영하기 너무 부담. 1인이 모두 운영해주면 그 자체가 차별화.

## 4. 패키지 설계 — 3-tier

### 가격 책정 원칙

- 사장 객단가가 39,000원 = 손님 5~10명분이 마케팅 대행비
- 사장이 단품 5개 따로 가입하면 SaaS+대행 합쳐 월 100~200만원 = 부담
- 1인 풀패키지로 묶으면 가격 압박 X + 사장 손이 떨어짐

### Lite — 월 39만 (예약·노쇼 방지 중심)

- 카카오 알림톡 자동발송 (예약확정/1일전/당일)
- 4주 재방문 알림톡 자동
- 영수증 후기 유도 메시지
- 월간 리포트 1장

→ **타깃**: 마케팅까지는 부담스럽고, 일단 노쇼·재방문만 잡고 싶은 사장. 진입 부담 최소.

### Standard — 월 79만 (마케팅 + 응대 자동화)

- Lite 전부 +
- 인스타 운영 (월 8회 + 릴스 2편) — 사장이 시술 사진만 카톡 전송
- 카카오 채널 GPT 자동응답 (자주 묻는 30Q + 디자인 컨설팅 답변)
- 신규 디자인 컨설팅 카드뉴스 (J/C/D컬 × 눈매 비교) 1세트 자동 제작 (P3 해결)

→ **타깃**: 신규 고객 유입까지 노리는 사장. **메인 패키지**.

### Pro — 월 129만 (검색·플레이스 풀커버)

- Standard 전부 +
- 네이버 플레이스 운영 (사진·메뉴·후기 응대 주 2회)
- 인스타 릴스 추가 (월 4편 → 6편)
- 시술 후 사진 AI 자동 보정 (눈은 사진 잘 안 받는 P4 해결)
- 월 1회 화상 컨설팅 (전월 데이터 기반 개선안)

→ **타깃**: 매출 800만원+ 월매출 도전하는 사장. 적은 수의 프리미엄 고객.

### 사장 입력 (모든 tier 공통)
**시술 사진만 카톡으로 전송. 끝.**

## 5. 월 200만원 달성 시나리오 (역산)

### 가능한 조합

| 시나리오 | 구성 | 합계 | 클라이언트 수 | 운영 부담 (주당 시간) |
|---|---|---|---|---|
| A | Lite 5곳 | 195만 | **5곳** | ~12h (관리만) |
| B | Standard 3곳 | **237만** | **3곳** | ~12h (콘텐츠 제작 포함) |
| C | Lite 2 + Standard 2 | 236만 | 4곳 | ~14h |
| D | Pro 1 + Standard 1 | 208만 | 2곳 | ~10h |

→ **추천: B (Standard 3곳)**. 단가 안정 + 클라이언트 수 적음 + 본인 노동 효율.
→ **백업: C** (Lite로 진입 → Standard 업셀).

### 6개월 로드맵

| 시점 | 마일스톤 | 누적 매출 | 주력 액션 |
|---|---|---|---|
| **M0 (5월)** | 시제품 1세트 완성 + 첫 5곳 발굴 + 첫 DM 발송 | 0 | 동네 속눈썹샵 검색·후보 리스트 |
| **M1 (6월)** | 첫 무료 베타 1곳 시작 (4주) | 0 | 시즌 정점 직전 push |
| **M2 (7월)** | 첫 유료 전환 시도 (Standard 79만) → **1건 = 79만** | 79만 | 시제품 5건 추가 발송 (시즌 효과) |
| **M3 (8월)** | 베타 2~3곳 운영, 유료 누적 **2건 = 158만** | 158만 | 시즌 정점, 후기·소개 적극 |
| **M4 (9월)** | 유료 누적 **3건 = 237만 → 200 돌파** | 237만 | 안정화 + 단가 검증 |
| **M5 (10월)** | 유료 3~4건, 첫 Pro 시도 | 250~330만 | 비시즌 진입, 단골 케어 강조 |
| **M6 (11월)** | 유료 4건 안정화 + Pro 1건 시도 = **300만+** | 300만+ | 다음 vertical 확장 검토 |

### 가정 (감사 가능)

- 시제품 → 베타 응답률: **40%** (5곳 발송 시 2곳 응답)
- 베타 → 유료 전환률: **40~50%** (Standard 가격 부담은 v3보다 낮음)
- 베타 → 소개 받기: **30~40%**
- 평균 베타 운영 4주 + 결과 보고 1주

→ M4 200만 달성은 **유료 3건만 있으면 됨**. 베타 8~10건 사이클로 6개월 이내 가능.

### 만약 안 되면 (백업)

| 상황 | 대응 |
|---|---|
| M3까지 베타 0건 | 시제품 quality 점검 + DM 톤 변경, 동네 확대 |
| M4까지 유료 1건만 | Lite 39만으로 진입 가격 낮춤 + 빠른 성과 입증 후 업셀 |
| M6까지 100만 미만 | 패키지 구조 재검토 + 인접 vertical (네일·왁싱) 동시 진입 |

| 인스타 캡션 5개 + 릴스 1편 (Before/After) + 사진 AI 보정 1장 | 2h | P4 | 즉시 사용 가능 |
| 알림톡 템플릿 3개 (예약확정 / 1일전 / 4주 재방문) | 0.5h | P2·P5 | 도입 즉시 노쇼 ↓ 입증 |
| 네이버 플레이스 진단 1장 (현재 후기·키워드 점검 + 개선 제안 3개) | 1h | P6 | 영업 무기 |
| 1장 종합 리포트 (현재 → 적용 시 예상 효과) | 0.5h | 전체 | DM 첫 인사 |

→ **DM 첫 메시지 (예시)**:

## 7. 첫 타깃 1곳 발굴 가이드

### Step 1 — 동네 후보 5~10곳 검색

1. **네이버 지도** "[동네명] 속눈썹연장" → 별점 4.0~4.7 + 후기 30~150건 (너무 많으면 이미 마케팅 대행 받는 곳)
2. **인스타 검색** "#[동네]속눈썹" → 팔로워 500~3,000 + 게시물 200~800 + 최근 7일 내 게시물 X 또는 1건만
3. **카카오 채널** 검색 → 운영 안 하거나 자동응답 X

### Step 2 — 1차 평가 (체크리스트)

| 항목 | 통과 |
|---|---|
| 1인 운영으로 보임 (게시물 1인칭 톤) | ✅ |
| 후기 50건 미만 또는 최근 1달 후기 X | ✅ (개선 여지 큼) |
| 인스타 게시물이 시술 사진만 + 캡션 짧음 | ✅ (콘텐츠 페인) |
| 네이버 플레이스 사진 5장 이하 | ✅ (P6 페인) |
| DM 응대가 빠르지 않음 (테스트 메시지로 확인) | ✅ (P1 페인) |

→ 5개 중 **3개 이상 통과 = 시제품 만들 1순위**.

### Step 3 — 시제품 제작 → DM 발송

- 1순위 1곳에 시제품 4~6h 만들고 DM
- 응답 24h 내 없으면 2순위로 이동
- 동시에 5곳 발송하지 말고 **1곳씩 시제품 quality 살리기** (첫 베타 확보가 가장 중요)

### Step 4 — 첫 베타 확보 후

- 무료 베타 4주 운영
- 매주 카톡 1줄 진척 공유
- 4주 종료 시 결과 1장 → 유료 전환 또는 소개 부탁

## 9. 진행 시 주의사항

- **시즌 의존**: 7~9월 정점, 11~2월 약함. 영업 푸시는 5~7월 집중.
- **알러지 응대**: 챗봇이 의료적 답변을 하지 않도록 프롬프트 가드 (내원·전문가 상담 안내로 회피).
- **단골 사적 관계**: 사장-단골 호감 관계가 매출 핵심. 자동화로 사장 톤 침해 X (메시지 톤·이모지 사장과 사전 합의).
- **카카오 알림톡 승인**: 비즈니스 채널 + 템플릿 사전 승인 필요. 첫 셋업에 1~3일 소요.
- **사진 보정 ethics**: AI 보정은 "또렷하게" 정도까지. 과도한 변형은 후기 신뢰 깎음.
