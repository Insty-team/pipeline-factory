## ⚠️ 카탈로그 본질 (먼저 읽고 시작)

**모든 글은 AI가 사장 톤으로 작성한 "게시 후보"**. 사장이 직접 게시한 기존 글이 아님.
2가지 분류로 표시:

| 분류 | 의미 | 신뢰도 |
|---|---|---|
| ♻️ **변형** | 사장 기존 글에 변화 줘서 비슷하지만 다른 글 (출처 logNo 명시) | ★★★ (사장 글 합성·재구성) |
| 🆕 **신규** | 판매 메뉴를 사장 톤으로 AI가 새로 구성한 글 | ★★ (메뉴 운영 사실 ✅, 글 본문 ❌) |

**점점 정확해지는 메커니즘**:
1. 베타 1주차 — 사장에 7편 카탈로그 보여주고 검수
2. 사장 "이 부분 바꿔주세요" → 즉시 정정 → 카탈로그 v2 갱신
3. 매주 사장 검수 통과한 글이 늘어남 → quality ↑
4. 4주차 끝나면 카탈로그 = "사장 검수 완료 글 모음" 자산화

### 사장 피드백 → 카탈로그 진화 워크플로우

```
사장 검수
   ↓
"이 단어 빼주세요" / "이 메뉴 가격 30% 할인 추가" / "메디핑크 표현 바꿔주세요"
   ↓
즉시 정정 → vN+1 카탈로그 갱신
   ↓
사장 검수 통과한 글 비율 증가
   ↓
4주 종료 시: 카탈로그 = "사장 검수 완료 글 모음" (운영 자산)
```

**점점 정확해지는 메커니즘**:
- 1주차: 신규 50% / 변형 50% — 사장 정정 빈도 높음
- 2주차: 사장 정정 반영본 게시 → 신뢰도 ↑
- 3주차: 새 시술 사진 받아 즉석 생성 → 사장 톤 학습 데이터 누적
- 4주차: 카탈로그 갱신 ≈ 90% 자동, 10% 사장 직접 수정
