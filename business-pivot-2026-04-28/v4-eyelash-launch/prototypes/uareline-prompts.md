### 0-A. 답글 (네이버 후기) — ~당·~욤·~용 압도적

| 어미 | 답글 58건 빈도 |
|---|---|
| **~당** | 34회 (1위) |
| **~욤** | 26회 (2위) |
| ~합니다 | 17회 |
| ~세요 | 15회 |
| **~용** | 13회 |
| ~드립니다 | 7회 |
| ~죠 | 4회 |
| ~드려요 | 3회 |

**원칙**: ~당·~욤·~용을 메인으로 사용 + ~합니다·~세요 보조. 한 답글에 어미 1~2개만 (반복 X).

### 0-B. 인스타 캡션 — 격식 톤 (~당·~욤 거의 X)

| 어미 | 인스타 31장 시각 추정 |
|---|---|
| **~합니다** | 다수 ("필수입니다!!") |
| **~네요** | 다수 ("예쁘네용~") |
| **~죠** | 가끔 ("이쁘죠~?") |
| **~로** / **~으로** | 다수 ("~로 마무리") |
| ~욤·~당 | **거의 X** |

**원칙**: 답글보다 격식. 시술 묘사 + 시그니처 멘트 위주. ~욤·~당 사용 X.
