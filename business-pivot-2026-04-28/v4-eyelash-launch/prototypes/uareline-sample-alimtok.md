## A. 예약 lifecycle 5종

### A1 — 예약 확정 (예약 완료 직후)

```
[유어라인] 예약 확정 안내드려요
김민지님, 예약이 확정되었습니다.

📅 5월 8일 (목) 14:00
💄 LED 연장 하프

방문 전날 다시 알림드릴게요~
변경/취소: 0507-1320-6511 또는 카톡 채널♡
```

### A2 — 1일 전 (방문 전날 18시)

```
[유어라인] 내일 예약 알림드려요
김민지님, 내일 14:00 예약 잊지 마세요!

💄 LED 연장 하프
📍 유어라인 (이수역 도보 5분)

오늘 컨디션 챙기시고 푹 주무세요~ 내일 뵐게요♡
변경 필요 시: 0507-1320-6511
```

### A3 — 당일 (방문 당일 9시)

```
[유어라인] 오늘 14:00 뵐게요
김민지님, 오늘 예약일이에요♡

💄 LED 연장 하프
📍 이수역 도보 5분

콘택트렌즈는 미리 빼주시고~
화장도 가볍게 해주시면 시술이 더 깔끔해요!

이따 뵙겠습니다.
```

### A4 — D+1 후기 요청 (시술 다음 날 오전)

```
[유어라인] 어제 시술 어떠셨어요?
김민지님, 어제 시술 잘 받으셨어요?

LED 연장 컬과 유지력 마음에 드신다면
네이버 후기 1줄만 남겨주시면 큰 힘이 됩니다♡

❗ 4시간의 법칙 잘 지키셨죠?
영양제 매일 발라서 결 관리해주세요~
```

### A5 — D+28 재방문 알림 (4주 후)

```
[유어라인] 한 달이 지났어요 김민지님♡
지난 시술 후 4주가 되었습니다!

💄 LED 연장 하프 - 리터치 추천 시기

지금 예약하시면 원하시는 시간 잡기 좋아요~
편한 시간 알려주시면 자리 안내드릴게요!

예약: 0507-1320-6511 또는 카톡 채널
```

## B. 신규 vs 단골 분기

### B1 — 첫 방문 환영 (A1 변형)

```
[유어라인] 첫 방문 환영합니다, 김민지님♡
예약이 확정되었습니다!

📅 5월 8일 (목) 14:00
💄 LED 연장 하프
📍 유어라인 (이수역 도보 5분)

[첫 방문 안내]
✔ 콘택트렌즈는 미리 빼주세요
✔ 눈 주변 화장은 가볍게~
✔ 시술 시간 약 60~90분 소요

방문 전날 다시 알림드릴게요~
궁금한 점은 카톡 채널로 편하게 물어보세요♡
변경/취소: 0507-1320-6511
```

### B2 — 5번째 단골 인사 (A1 변형)

```
[유어라인] 5번째 방문 감사드려요, 김민지님♡
예약이 확정되었습니다!

📅 5월 8일 (목) 14:00
💄 LED 연장 하프

✨ 단골 회원권 안내
5번째 방문이시면 회원권 가입 시
다음 시술 10% 할인 + 영양제 무료 증정♡

자세한 안내는 시술 시 도와드릴게요~
변경/취소: 0507-1320-6511
```

### B3 — 10번째 단골 특별 (A1 변형)

```
[유어라인] 10번째 방문, 정말 감사드려요 김민지님♡
예약이 확정되었습니다!

📅 5월 8일 (목) 14:00
💄 LED 연장 하프

🎁 10회차 작은 선물
시술 시 영양제 듬뿍 넣어드릴게요~
EYE 클리닉케어 + 샴푸서비스 무료♡

오랫동안 함께해주셔서 진심으로 감사드립니다.
변경/취소: 0507-1320-6511
```

## C. 예외 시나리오

### C1 — 노쇼 후 부드러운 follow-up (노쇼 발생 당일 저녁)

```
[유어라인] 오늘 못 오셨네요 김민지님
혹시 무슨 일 있으셨어요?

급한 일이셨다면 다음에 편하게 다시 잡아주세요~
저희 단골이시니 부담 없이 연락주세요♡

다음 예약 잡으시려면:
0507-1320-6511 또는 카톡 채널
```

> ⚠️ 노쇼 후 부드러운 톤이 중요. 비난 X, 다음 방문 유도.

### C2 — 예약 변경 요청 응답

```
[유어라인] 예약 변경 안내드려요
김민지님, 요청하신 일정으로 변경되었습니다.

📅 변경 전: 5월 8일 (목) 14:00
📅 변경 후: 5월 10일 (토) 11:00
💄 LED 연장 하프

변경된 일정 다시 1일 전 알림드릴게요~
감사드립니다♡
```

### C3 — 휴무 안내 (사장 일정 1주 전)

```
[유어라인] 휴무 안내드려요
안녕하세요, 유어라인입니다.

📅 5월 15일 (목) ~ 5월 17일 (토) 휴무
📅 5월 18일 (일) 정상 영업

휴무 전 시술 받으실 분은 미리 예약 부탁드립니다~
양해해주셔서 감사드립니다♡

예약: 0507-1320-6511 또는 카톡 채널
```

## D. 추가 매출 알림 (M9 활용)

### D1 — 메디핑크 단골 추천 (메디핑크 비대상 단골에게)

```
[유어라인] 김민지님께 [메디핑크] 알려드려요♡
늘 찾아주시는 김민지님께
새로 들어온 [메디핑크]를 소개해드릴게요!

✨ 멜라닌 색소 케어 (유륜·외음부·바디)
✨ 통증 없는 1:1 시술
✨ 28일의 기적

🎁 단골 한정: 펌 무료 서비스권 증정

자세한 상담은 다음 방문 시 또는
카톡 채널로 편하게 물어봐 주세요~

상담: 0507-1320-6511 또는 카톡 채널
```

### D2 — 영양제 재구매 안내 (영양제 구매 후 6주)

```
[유어라인] 영양제 다 쓰셨을 시기네요♡
김민지님, [펑키 블랙 영양제] 받으신 지
6주가 되어가요~

💧 영양제는 꾸준한 사용이 핵심!
지금쯤 다 쓰셨다면 결 관리 위해
재구매 추천드려요.

✨ 단골 할인가로 만나보실 수 있어요!

다음 방문 시 또는 카톡으로 알려주시면
바로 준비해드릴게요♡

문의: 0507-1320-6511 또는 카톡 채널
```

### D3 — 이탈 위험 단골 안부 (6주+ 미방문)

```
[유어라인] 김민지님~ 한동안 안 보이셔서♡
잘 지내고 계세요?

벌써 마지막 시술 후 6주가 지났네요~
혹시 다른 일정이 바쁘셨나요?

영양제 잘 챙기고 계시는지 궁금해서
안부 인사드려요♡

다음 시술 일정 잡으시면 미리 알려드릴게요~
편한 시간 알려주시면 자리 잡아드립니다.

예약: 0507-1320-6511 또는 카톡 채널
```
