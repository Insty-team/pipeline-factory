## 핵심 발견: 즉시 영업 가능한 3곳

### 1. 메티의원 (meticlinic.com) — 최우선 타겟
- **왜**: Schema 0, 메타태그 0, JS 의존, 연락처조차 HTML에 없음
- **영업 포인트**: "사장님, 홈페이지가 imweb으로 만들어져 있어서 AI가 아예 못 읽습니다. 시술 정보가 있어도 AI한테는 빈 페이지나 마찬가지예요."
- **경쟁사 비교**: 같은 강남역 모리스의원은 Schema 있음 → "경쟁사는 이미 하고 있습니다"
- **연락**: 홈페이지에서 카카오/전화번호 확인 필요 (JS 렌더링이라 직접 방문해서 확인)

### 2. 유스타일나인 (ustyle9.com) — 강력 타겟
- **왜**: Schema 0, robots.txt에 구글봇만 허용 (GPTBot/ClaudeBot 차단!)
- **영업 포인트**: "사장님, 지금 robots.txt 설정이 구글봇만 허용이라 ChatGPT, Gemini, Perplexity가 사이트에 접근 자체를 못 합니다. 완전 차단 상태예요."
- **연락**: 02-553-2011

### 3. 세진피부과 (sejinskin.com) — 극단적 사례
- **왜**: robots.txt에 네이버 Yeti만 허용 = AI 크롤러 전부 차단
- **영업 포인트**: "사장님, 지금 네이버 검색봇만 접근 가능하고 나머지 전부 차단입니다. AI 검색에는 사이트가 존재하지 않는 것과 같습니다."
- **연락**: 홈페이지에서 확인 필요

## 아웃리치 메시지 (중소 클리닉용 최종본)

### 2026-04-20 업데이트 — 오늘은 유스타일나인 1곳만 테스트

목표는 계약이 아니라 `무료로 짧게 보내주세요` 또는 `무슨 문제인가요?` 반응을 얻는 것. 첫 메시지에서는 `GEO`, `AEO`, `schema`, `llms.txt` 단어를 쓰지 않는다.

#### 유스타일나인용 — 공식 정보 운영 프레임
```text
안녕하세요. AI 검색에서 병원 정보가 어떻게 보이는지 확인하다가 연락드립니다.

요즘 고객들이 ChatGPT나 Gemini로 “강남 레이저 제모 추천”처럼 검색하는데,
유스타일나인의원은 관련 검색에서 잘 잡히지 않는 편입니다.

확인해보니 홈페이지 설정상 AI가 병원 정보를 읽기 어려운 부분이 있어 보여서요.
정확히 말하면 “노출을 늘리는 작업”이라기보다, AI가 병원 공식 정보를 헷갈리지 않게 만드는 작업입니다.

원하시면 무료로 아주 짧게 현황만 정리해서 보내드릴게요.
```

## 아웃리치 실행 순서 (내일부터)

### Day 1 — 연락처 확보 + 1차 DM
1. 메티의원 카카오톡/전화번호 확인 (홈페이지 직접 접속)
2. 유스타일나인 → 02-553-2011 / 카카오톡 확인
3. 세진피부과 연락처 확인
4. 3곳 동시에 카카오톡 DM 발송
5. ChatGPT "강남 레이저 제모 추천" 스크린샷 첨부

### Day 2 — 팔로업 + 추가 타겟
6. 무응답 시 전화 팔로업
7. 강남언니/네이버플레이스에서 추가 5곳 발굴
8. 각 클리닉 robots.txt만 빠르게 체크 (curl로 30초)
9. 차단된 곳 우선 DM

### Day 3~7 — 반응 기록 + 확장
10. 반응을 4가지로 분류: 무응답/거절/관심/URL전달
11. "관심" 이상이면 즉시 감사 리포트 제작 (1시간)
12. "URL전달"이면 당일 구조화 데이터 초안 작성
13. 누적 10곳 이상 아웃리치 완료 목표

## 세일즈 퍼널 정리

```
[1단계] 리드 발굴
├── 네이버플레이스/강남언니에서 리뷰 50~300개 클리닉
├── 홈페이지 robots.txt 체크 → AI봇 차단된 곳 우선
└── ChatGPT에서 클리닉명 검색 → 안 나오면 타겟 확정

[2단계] 1차 접촉 (카카오톡 DM)
├── "사장님 클리닉 ChatGPT에 안 나옵니다" + 스크린샷
├── "무료 분석 리포트 보내드릴까요?"
└── 목표: 답변 유도

[3단계] 무료 감사 리포트 전달
├── GEO 점수 + 경쟁사 비교
├── "이래서 안 나오는 겁니다" 문제 진단
├── "이렇게 하면 나올 수 있습니다" 해결책 제시
└── 목표: "얼마예요?" 질문 유도

[4단계] 유료 전환
├── 셋업 30~50만원 제안
├── "robots.txt + Schema + llms.txt + 프로필 페이지 + before/after"
├── 1~3일 작업, 승인 1회
└── 목표: 결제

[5단계] 월정액 전환
├── 셋업 완료 2주 후 "변화 리포트" 전달
├── "계속 모니터링 + 월간 리포트 월 15만원"
└── 목표: 장기 관리 계약
```
