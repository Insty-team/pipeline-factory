## 인수 대상이 되려면

### SEO/마케팅 업계 M&A 패턴

- Semrush → Search Engine Land 인수 (콘텐츠+데이터)
- 대형 광고사들이 디지털 에이전시를 인수하는 패턴 반복
- **인수되는 회사의 공통점**: 고객 기반 + 독자 기술/데이터 + 반복 수익

### 누가 우리를 사고 싶어할까?

| 잠재 인수자 | 이유 |
|------------|------|
| 기존 병원 대행사 (하룹, MPR, 금솔) | GEO 역량 내재화하고 싶지만 직접 만들기 어려움 |
| 대형 광고사 (TBWA, 제일기획 등) | 헬스케어 버티컬 GEO 전문성 필요 |
| SaaS 회사 (Geoptie 등) | 고객 기반 + 운영 노하우 확보 |
| 해외 GEO 에이전시 | 한국 시장 진출 교두보 |

### 인수 가치를 높이는 5가지 자산

**1. 반복 수익 (Recurring Revenue)**
- 월정액 관리 고객 수가 핵심 밸류에이션 지표
- 에이전시 인수는 보통 연 매출의 3~7배
- 월 관리 20건 × 20만원 = 월 400만 = 연 4,800만
- 밸류에이션: 1.4억~3.4억

**2. 버티컬 전문성 (Vertical Expertise)**
- "한국 의료미용 GEO" = 좁지만 깊은 전문성
- 지오랭크는 범용, 우리는 뷰티/클리닉 특화
- 특화 = 프리미엄 = 인수 시 더 높은 배수

**3. 독자 데이터/도구**
- 클리닉별 GEO 벤치마크 데이터 축적
- AI 검색 모니터링 자동화 시스템
- before/after 증거 아카이브
- → 이게 쌓이면 "데이터 자산"이 됨

**4. 고객 관계**
- 클리닉 사장님과의 직접 관계
- 재계약률이 높을수록 가치 상승
- 화이트라벨 대행사 파트너십도 관계 자산

**5. 프로세스/시스템**
- 감사→셋업→모니터링이 자동화되어 있으면
- 인수자 입장에서 "사람 없이도 돌아가는 시스템" = 높은 가치

## A→B→매각 로드맵

### Phase A: 직접 영업 (월 1~3)
```
목표: 파일럿 5건, 유료 3건
├── 중소 클리닉 직접 아웃리치
├── 무료 감사 → 셋업 → 월정액 전환
├── before/after 사례 축적
└── 작업 프로세스 문서화 + 자동화 시작
```

### Phase B: 대행사 파트너십 (월 3~6)
```
목표: 대행사 2곳과 파트너십, 관리 15건+
├── 실적 들고 하룹/MPR/금솔에 제안
├── "GEO를 기존 서비스에 추가하세요, 저희가 화이트라벨로"
├── 대행사당 5~10건 클리닉이 열림
└── 관리 건수 급증 → 자동화 필수
```

### Phase C: 시스템화 + 데이터 축적 (월 6~12)
```
목표: 관리 30건+, 자동화 80%, 데이터 자산 구축
├── 감사/모니터링/리포트 완전 자동화
├── 클리닉 GEO 벤치마크 DB 구축
├── 버티컬 확장 (피부과 → 치과 → 성형외과)
└── 월 수익 500만+ (관리 30건 × 15~20만)
```

### Phase D: 매각 or 확장 (12개월+)
```
매각 시나리오:
├── 연 매출 6,000만+ (관리 30건)
├── 밸류에이션 1.8억~4.2억 (3~7배)
├── 인수 대상: 대형 대행사 or SaaS
└── 또는: 계속 운영하며 연 1억+ 수익

확장 시나리오:
├── SaaS 전환 (셀프서브 GEO 감사 도구)
├── 해외 진출 (일본 미용클리닉)
└── 교육/강의 (병원 마케터 대상 GEO 교육)
```
