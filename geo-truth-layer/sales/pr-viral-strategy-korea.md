## 바이럴이 터지는 콘텐츠의 4가지 심리 트리거

1. Social Currency — 공유하면 나도 똑똑해 보이는 콘텐츠
2. Emotional Activation — 놀람/분노/흥분 등 강한 감정
3. Practical Value — 바로 쓸 수 있는 팁/정보
4. Identity Signaling — "나도 이런 사람" 표현

## 2026 바이럴 공식: 3초 안에 잡고, 공유/저장하게 만들기

### 알고리즘이 밀어주는 신호 (2026)

| 신호 | 가중치 |
|------|--------|
| 공유 (DM 전송) | 최고 |
| 저장 | 매우 높음 |
| 댓글 | 높음 |
| 시청 완료율 (70%+) | 높음 |
| 좋아요 | 보통 |

좋아요보다 공유/저장이 핵심. "이거 너도 봐봐"하고 보내고 싶은 콘텐츠.

### 7가지 훅 패턴

1. 반상식: "매일 포스팅하면 오히려 손해입니다"
2. 약속: "이 영상 끝까지 보면 X를 알게 됩니다"
3. 액션 시작: 설명 없이 결과부터 보여주기
4. 질문: "왜 강남 클리닉 90%가 AI 검색에 안 나올까?"
5. 시각 충격: before/after 변환, 숫자 대비
6. 개인 투자: "6개월 직접 테스트해봤습니다"
7. 트렌드 활용: 기존 인기 주제에 내 관점 얹기

### 가장 잘 되는 콘텐츠 형태 (2026)

1. before/after 변환 (최강)
2. 미신 깨기 + 증거
3. 타임랩스/프로세스
4. 저가 vs 고가 비교
5. 문제 해결 튜토리얼
6. Top 5 랭킹
7. 실험 결과 공개

## Sam 계정에 최적화된 전략

### 왜 숏폼 영상이 텍스트보다 나은가

| | 텍스트 (X/블루스카이) | 숏폼 영상 (쇼츠/릴스) |
|---|---|---|
| 최대 도달 | 수천~수만 | 수십만~수백만 |
| 3초 후킹 | 어려움 | 쉬움 (시각+청각) |
| 공유 확률 | 낮음 | 높음 (DM 공유) |
| 알고리즘 부스트 | 약함 | 매우 강함 |
| 제작 시간 | 10분 | 30분~1시간 |
| 얼굴 노출 | 불필요 | 불필요 (화면녹화/자막) |

### 얼굴 안 나오는 숏폼 만드는 법

화면녹화 + 자막 + 나레이션(선택):
- ChatGPT 화면 녹화하면서 "강남 제모 추천" 검색
- AI가 뭐라고 답하는지 실시간 보여줌
- 자막: "강남 클리닉 200곳 중 AI가 아는 곳은 3곳뿐"
- 끝에: "왜 이런지 다음 영상에서"

이러면 얼굴 안 나오고, 30분이면 만들 수 있고, 후킹 강력.

## 채널별 역할 분담

### 1순위: 유튜브 쇼츠 (도달 최대화)
- 한국인 87%가 쇼츠 시청
- 얼굴 없이 화면녹화+자막으로 가능
- 주 3~5개 (같은 소재 각도만 바꿔서)
- 목표: 조회수 + 구독자

### 2순위: 인스타 릴스 (동시 업로드)
- 쇼츠와 같은 영상 그대로 올림 (시간 추가 0)
- 인스타는 DM 공유가 알고리즘 핵심
- 목표: 공유 + 저장

### 3순위: X (텍스트 + 영상 링크)
- 영상 올린 후 핵심 인사이트만 텍스트로
- 마케터/창업자 커뮤니티 타겟
- 스레드(긴 글) 형태도 바이럴 가능
- 목표: 전문가 인지도 + DM 문의

### 4순위: 아이보스 (병원 마케팅 커뮤니티)
- 영상/글 링크 공유 + 전문가 답변
- 직접 영업 채널
- 목표: 리드 확보

## 바이럴 터지는 영상 만드는 공식

### 구조 (60초 이내)

0~3초: 후킹 (반상식 or 질문 or 충격 숫자)
3~15초: 문제 설명 (공감)
15~45초: 핵심 내용 (before/after or 방법)
45~60초: CTA ("팔로우하면 다음 실험도 공유")

### 제작 도구 (무료~저가)

- 화면 녹화: OBS (무료)
- 자막: CapCut (무료), VREW (한국어 자동자막)
- 썸네일: Canva (무료)
- 편집: CapCut (무료)
- 소요시간: 영상 1개당 30~45분

글 -> 팔로워 -> DM -> 감사 -> 셋업

### 복합 전략 (최적)
숏폼 영상 (도달) + X 텍스트 (심화) + 아이보스 (직접 리드)
= 넓게 뿌리고 + 깊게 설명하고 + 직접 잡기

## 핵심 인사이트

1. 텍스트 글 하루 1개보다 숏폼 영상 주 3~5개가 도달 100배
2. 얼굴 안 나와도 됨 — 화면녹화+자막이면 충분
3. 같은 소재를 여러 각도로 재활용 (1소재 = 3~5영상)
4. 3초 후킹이 전부 — 나머지는 부차적
5. 공유/저장을 유도하는 구조 > 좋아요 유도
6. 바이럴은 운이 아니라 시스템 — 많이 올릴수록 터질 확률 올라감
