## 반복 템플릿

### 고정 구조 (매 영상 동일)

[0~3초]  "오늘도 AI한테 물어봤습니다."
[3~15초] 오늘의 질문/실험 대상 소개
[15~45초] 실험 과정 (화면녹화 + 자막)
[45~53초] 결과 + 한 줄 인사이트
[53~55초] "...인데, 사장님은 모릅니다."
→ 자연스럽게 처음으로 루핑

캡션: 핵심 한 줄 + 해시태그
핀댓글: 참여 유도 질문

### 왜 이 템플릿인가

1. "오늘도 AI한테 물어봤습니다" — AI가 도구, 내가 주체. 실험이 바뀌어도 성립.
2. 실험 소재만 교체 — 포맷은 고정, 제작 부담 최소.
3. "사장님은 모릅니다" — AI↔사업자 간극이 핵심이면 영원히 성립. 밈화 가능.
4. 아무 편이나 봐도 이해 — 진입장벽 0.
5. 루핑 구조 — 재시청 유도, 조회수 자연 증가.

### 이 템플릿이 커버하는 실험들

| 현재 실험 | 예시 영상 |
|----------|----------|
| GEO (클리닉) | "오늘도 AI한테 물어봤습니다. 강남 제모 추천해줘." |
| GEO (식당) | "오늘도 AI한테 물어봤습니다. 을지로 파스타 맛집 추천해줘." |
| 자동화 | "오늘도 AI한테 물어봤습니다. 이 반복 업무 자동화 돼?" |
| 리뷰 분석 | "오늘도 AI한테 물어봤습니다. 이 가게 리뷰 300개 분석해줘." |
| 무엇이든 | "오늘도 AI한테 물어봤습니다. [질문]" |

실험이 실패해도 "이번 실험은 안 됐습니다...인데, 사장님은 모릅니다" → 실패도 콘텐츠.

- 영상 링크 자연스럽게 공유
- 직접 영업 채널

### 나중에: LinkedIn, 브런치, 틱톡

## 팬이 생기는 구조

### 바이럴 (새 시청자 유입)
- 각 영상이 독립적으로 후킹 → 아무 편이나 바이럴 가능
- 숫자/데이터 기반 → 공유 욕구 ("이거 너도 봐봐")
- "사장님은 모릅니다" → 밈화 → 댓글에서 자연 확산

### 팬 전환 (재방문 유도)
- 반복 템플릿 → 3~4편 보면 패턴 체화 → 익숙함 → 재방문
- 실험이 매번 다름 → "이번엔 뭘 했을까?" 궁금
- 댓글 참여 ("다음에 뭐 실험할까요?") → 소속감
- 댓글 답변 + 반영 → 1:1 소통 느낌 → 충성

### 팬 → 고객 (자연 전환)
- 사장님이 직접 봄 → "나도 해봐야 하나?" → DM
- 사장님 지인이 봄 → "사장님 이거 보셨어요?" 공유
- 전문가 인지도 → "이 사람한테 맡기면 되겠다"

## 콘텐츠 운영

### 소재 3가지 카테고리

| 카테고리 | 비율 | 예시 |
|---------|------|------|
| 실험 결과 | 60% | "AI한테 물어봤더니 이 결과가" |
| 인사이트 | 25% | "AI 시대에 사업자가 모르는 것 3가지" |
| 비하인드 | 15% | "이번 실험 실패한 이유", "사장님 반응" |

### 소재가 고갈되지 않는 이유
- 매일 하는 실험이 곧 소재 (따로 만들 필요 없음)
- 실패도 소재 ("이건 안 됐습니다...인데, 사장님은 모릅니다")
- 시청자 요청도 소재 ("댓글에서 요청한 OO 해봤습니다")
- 업종을 바꾸면 무한 소재 (클리닉 → 식당 → 쇼핑몰 → ...)

### 하루 시간표 (30~45분)

| 시간 | 작업 |
|------|------|
| 15분 | 오늘 실험한 것 → 화면녹화 + 자막 편집 |
| 10분 | 쇼츠 + 릴스 + X 동시 업로드 |
| 5분 | 댓글 확인 + 답변 2~3개 |
| 5분 | 다음 소재 메모 |

### 제작 도구 (전부 무료)
- 화면 녹화: OBS
- 자막/편집: CapCut 또는 VREW (한국어 자동자막)
- 썸네일: Canva
- 업로드: 직접 (자동화는 나중에)

## PR → 수익 연결 경로

### 직접 전환
영상 시청 → "나도 AI 검색에 안 나오나?" → DM → 무료 감사 → 유료

### 간접 전환 (더 강력)
영상 시청 → 사장님 지인이 공유 → "이 사람한테 물어봐" → DM → 유료

### 장기 전환 (가장 강력)
영상 50개 쌓임 → "AI로 사업자 돕는 사람 = Sam" 인식
→ 세미나 초대, 언론 기고, 대행사 제안 → 채널이 알아서 열림

### 실험 바뀌어도 작동
GEO 안 되면 → 다른 실험 시작 → 같은 템플릿으로 영상 계속
→ 팬은 "Sam이 이번엔 뭘 하나" 보러 옴
→ 새 실험의 첫 고객이 기존 팬에서 나올 수 있음
