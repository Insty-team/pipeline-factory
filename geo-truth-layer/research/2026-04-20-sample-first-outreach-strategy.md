# 2026-04-20 샘플 선제공 전략 — 대행사식 콜드 어프로치 적용

## 핵심 인사이트

일러스트 대행자가 “미리 이런 스타일로 만들어봤는데 필요하면 완성해드릴게요”라고 접근하듯, AI 공식 정보 운영 서비스도 말로 설명하기보다 **아주 작은 결과물**을 먼저 보여주는 편이 반응을 만들 가능성이 높다.

현재 첫 메시지는 예의 있고 안전하지만, 병원 입장에서는 아직 추상적이다.

- “AI가 병원 정보를 읽기 어렵다”가 무슨 뜻인지 바로 보이지 않음
- “무료 현황 메모”도 대행사 영업처럼 느껴질 수 있음
- 읽었지만 답이 없는 이유는 관심이 없어서라기보다, 답할 만큼 구체적이지 않았을 가능성이 있음

따라서 다음 콜드 어프로치에서는 permission-first를 유지하되, **작은 샘플을 더 빨리 보여주는 전략**을 테스트한다.

## 원칙

### 무료로 줄 것

- 문제 1개
- 캡처 1장
- 3줄 현황 메모
- AI용 공식 정보 카드 샘플 1개
- Before/After 구조 예시 1개

### 무료로 주지 않을 것

- 전체 GEO/AEO 리포트
- 전체 구조화 데이터
- 전체 페이지 제작
- 전체 경쟁사 비교
- 구체 실행 파일 전체

## 내일 신규 클리닉 연락 전략

### 첫 메시지 구조

기존:

```text
무료로 짧은 현황 메모를 전달드려도 될까요?
```

개선:

```text
짧게 확인해보니 AI가 병원 정보를 헷갈릴 수 있는 지점이 1개 보여서요.
내용이 길지 않아 3줄로만 남겨드려도 될까요?
```

또는 더 선제공형:

```text
길지 않아 3줄만 남기겠습니다. 필요 없으시면 답변 안 주셔도 괜찮습니다.
```

## 신규 클리닉용 첫 메시지 v2

```text
안녕하세요. 병원/클리닉 정보가 ChatGPT 같은 AI 검색에서 어떻게 보이는지 테스트하고 있는 Sam입니다.

[클리닉명] 정보를 확인하다가, AI가 병원 공식 정보를 헷갈릴 수 있는 지점이 하나 보여 연락드렸습니다.

제가 뭔가를 바로 판매하려는 건 아니고,
AI가 병원 정보를 정확히 읽게 만들 수 있는지 테스트하고 있습니다.

내용이 길지 않아서 3줄 현황 메모로만 남겨드려도 될까요?
혹시 환자 상담 채널이라 전달이 어려우면 여기서 마무리하겠습니다.
```

## 신규 클리닉용 선제공 팔로업 v2

첫 메시지를 읽고 답이 없으면 24~48시간 후 1회만 보낸다.

```text
안녕하세요. 어제 AI 검색 관련해서 짧게 문의드렸던 Sam입니다.

혹시 전달이 어려운 채널이면 여기서 마무리하겠습니다.
다만 내용이 길지 않아 3줄만 남기겠습니다.

1. ChatGPT/Gemini에서 “[검색어]”처럼 찾을 때 [클리닉명] 정보가 잘 잡히지 않는 편입니다.

2. 홈페이지 정보는 있지만 AI가 병원명, 주요 시술, 위치/상담 정보를 한 번에 이해하기 어려운 구조로 보입니다.

3. 제가 보는 방향은 광고를 더 하자는 게 아니라, AI가 병원 공식 정보를 헷갈리지 않게 읽도록 정리하는 작업입니다.

필요 없으시면 답변 안 주셔도 괜찮습니다.
```

## AI 공식 정보 카드 샘플

답장이 오거나 관심을 보이면 아래처럼 “우리가 만드는 결과물”을 1개만 보여준다.

```text
예를 들면 AI가 [클리닉명]을 설명할 때는 이런 식의 공식 정보 카드가 필요합니다.

[클리닉명]은 [지역]의 [진료/시술 분야] 클리닉입니다.
주요 정보는 [대표 시술/상담 분야]이며,
방문 전 가격, 이벤트, 시술 가능 여부는 공식 홈페이지나 상담 채널에서 확인하는 것이 좋습니다.
```

주의:

- 의료 효과/시술 가능 여부를 단정하지 않는다.
- 가격·이벤트·의료진 정보는 공식 출처 확인 전 단정하지 않는다.
- “추천 병원”처럼 광고/의료법상 민감할 수 있는 표현은 피한다.

## E-001에 적용할 테스트 설계

다음 신규 클리닉 1곳에는 기존 메시지와 다른 v2를 사용한다.

- v1: “무료 현황 메모 전달드려도 될까요?”
- v2: “AI가 헷갈릴 수 있는 지점 1개가 보여 3줄로 남겨드려도 될까요?”

측정:

- 읽음 여부
- 답장 여부
- 답장 유형: `보내주세요 / 어떤 내용인가요 / 담당자 연결 / 거절 / 무응답`
- v1 대비 답장률이 나아지는지

## 다음 액션

내일 신규 클리닉에 연락할 때는 3곳 이상 보내지 않는다.
추천은 **1곳만 v2로 테스트**한다.

후보:

1. 메티의원 — 기존 Tier 1이나 연락 채널 확인 필요
2. 추가 강남/압구정 피부·제모 클리닉 1곳 — 카카오 채널이 바로 확인되는 곳 우선

선택 기준:

- 카카오톡 채널이 공식 홈페이지에서 바로 확인됨
- 홈페이지에 시술 정보는 있으나 AI가 읽기 어려워 보임
- 병원/클리닉 규모가 너무 크지 않아 담당자 반응 가능성이 있음
