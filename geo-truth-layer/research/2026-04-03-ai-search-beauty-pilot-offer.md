# AI Search Beauty Discovery — Pilot Offer

Date: 2026-04-03

## One-line value proposition
광고를 하나 더 하자는 게 아니라, 기존 홈페이지/가격표 정보를 AI가 정확하게 읽고 비교할 수 있게 공식적으로 구조화해드리는 서비스입니다.

## Short version
- You give us official URLs.
- We extract only source-backed facts.
- We publish one structured public profile page optimized for crawlability and AI readability.
- You approve before it goes live.
- We track whether the page gets indexed, clicked, or cited better than before.

## Primary promise
We do **not** promise guaranteed ranking.
We promise:
- clearer, source-backed structured representation
- lower distortion risk in AI/search discovery
- a measurable before/after testing process

## Who this is for first
- laser hair removal clinics
- clinics with public pricing pages
- clinics with multiple branches or mixed language audiences
- high-consideration beauty providers where comparison matters

## Pilot package (v1)
### Included
1. Source audit of 1–5 official URLs
2. Fact extraction and normalization
3. One public structured profile page
4. Source attribution section
5. Structured metadata / crawlability checks
6. One approval round
7. Baseline + follow-up AI-search test log
8. CTA click tracking for booking/contact page

### Not included in pilot
- full site rebuild
- generic SEO campaign
- ad management
- guaranteed placement in ChatGPT/Gemini/Google AI features
- medical copywriting beyond source-backed restructuring

## Suggested pricing test
### Option A — low-friction pilot
- KRW 150,000
- one page, one source set, one approval round

### Option B — standard pilot
- KRW 300,000
- one page + one update round + 30-day check-in

### Option C — multi-page / branch-ready
- KRW 500,000
- up to 3 pages for one clinic / branch / treatment cluster

## Success metrics for the pilot
Provider-facing metrics:
- was the page approved quickly?
- did the provider say the page is more accurate/useful than scattered sources?
- did they request more pages?

Discoverability metrics:
- indexed or not
- bot-accessible or not
- before/after prompt-set change
- referral clicks / booking clicks

## Talk track
### Core pitch
"지금 하시던 홍보를 바꾸라는 게 아닙니다. 이미 있는 공식 정보를 바탕으로, 고객과 AI가 헷갈리지 않게 읽을 수 있는 한 장짜리 공식 프로필 레이어를 깔아드리는 겁니다."

### If they say “우린 홈페이지 있는데요?”
"맞아요. 홈페이지가 있어도 정보가 여러 페이지에 흩어져 있거나, 가격/지점/시술 적합성이 비교형 질문에 바로 안 잡히는 경우가 많습니다. 저희는 홈페이지를 대체하는 게 아니라, AI 검색과 비교 상황에서 더 잘 읽히는 정리 레이어를 만듭니다."

### If they say “SEO 업체랑 뭐가 다른가요?”
"일반 SEO처럼 광범위한 랭킹 작업이 아니라, 기존 공식 정보를 source-backed structured page로 만드는 데 초점을 둡니다. 핵심은 트래픽 약속보다 정확성과 비교 가능성입니다."

### If they say “AI 검색에 진짜 뜨나요?”
"노출을 보장하진 않습니다. 대신 공개적으로 크롤 가능한 구조, structured data, source attribution, fixed-query before/after 로그까지 갖춰서 ‘더 잘 읽히고 덜 왜곡되게’ 만드는 쪽을 검증합니다."
