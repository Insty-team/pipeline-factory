# YouTube 운영 클리닉 GEO 점수 감사

> 작성일: 2026-04-26
> 가설: 유튜브 운영 클리닉도 schema·AI 검색 최적화는 별개 도메인이라 점수 낮을 것 → 우리 outreach 메인 풀로 격상 가능?

## 결론 한 줄

**일부 뒷받침** — 13곳 평균 43점으로 기존 타겟(~15점)보다 훨씬 높지만, 4곳은 0~30점대로 즉시 영업 가능한 "낮게 매달린 과일"이며 유튜브 활성도까지 갖춰 응답 확률↑.

## 결과 요약
- 조사 클리닉 13개 (피부과 7, 성형외과 6 — 모두 자체 도메인+활발한 YouTube 보유)
- 평균 GEO 점수: **43/100**
- 기존 타겟 평균(메티/유스타일/세진): ~15/100 → 신 후보풀이 평균적으론 약 3배 더 정돈됨

- 점수 50점 이상 (영업 부적합): **6개**

## 영업 적합 후보 Tier 1

### 1. 윤앤정성형외과 (hiynj.com) — 25점
- 진입 포인트: **HTTPS 자체가 없다.** 443 포트 미구현이라 Chrome 주소창 빨간 경고. AI 검색은 https URL 우선이라 사실상 비가시.
- 윤인대 원장은 성형외과의사회 부회장이며 "성형할 사람·성형하지 말아야 할 사람" 영상이 40만 뷰 — 디지털 인지도와 사이트 인프라 격차가 극심.

### 2. 앤드성형외과 (andps.co.kr) — 0점
- 진입 포인트: **사이트 자체가 모든 크롤러에게 406 Not Acceptable + self-signed cert.** GPTBot/ChatGPT/Claude 어느 누구도 페이지에 도달 못 함. 영업 메시지: "ChatGPT가 귀 클리닉을 답변할 수 없는 이유 — 30초 입증" 가능.
- 이미 andps-global.co.kr이라는 별도 도메인은 정상 → 메인 도메인만 재설정하면 즉시 회복됨(영업 ROI 명확).

### 3. 삼성화이트피부과 (samsungwhitederma.com) — 0점
- 진입 포인트: 앤드와 동일 패턴. **OEM "My Company Ltd" 기본 SSL cert + 406** = 사이트가 셋업 후 마무리 안 됨. 박현정 원장 닥터화이티TV 활발.

### 4. 루트테마피부과 (themeskin.kr) — 30점
- 진입 포인트: SSL cert가 lasik4u.co.kr 도메인용으로 발급된 self-signed → 브라우저 경고 + AI 봇 무시. EUC-KR 인코딩 + JSON-LD 0 + 키워드 나열 desc로 GEO 인프라 전혀 없음. 4인 전문의 채널 활발.

## 영업 부적합 (제외)
- VOS 80 / 창피부과 70 / 위드윈 65 / WJ원진 65: yoast·schema가 이미 깔려있어 우리 코어 제안과 가치 중복.
- 바노바기 55, 컴포트 50: 50점대 경계 — schema 0개 또는 부적합 1개라 잠재적 보충 영업은 가능하나 우선순위는 낮음. 메티(15)·유스타일(20) 같은 압도적 부재가 아님.
