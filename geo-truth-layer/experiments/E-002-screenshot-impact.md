# E-002: 스크린샷 첨부 vs 미첨부 반응 비교

## 메타

- 상위 가설: **H-MKT** (시장) + **H-MSG** (메시지)
- 상태: `PLANNED`
- 설계일: 2026-04-04
- 선행 조건: E-001 Pass (도달 가능 확인)

---

## 가설

> ChatGPT 실제 검색 스크린샷("사장님 병원 안 나옵니다")을 첨부하면,
> 텍스트만 보낸 것보다 응답률이 높다.

> 부가 가설: 스크린샷을 보면 사장님이 AI 검색 노출을 실제 문제로 인식한다.

---

## 실험 설계

### 조건
| 그룹 | 메시지 | 첨부 |
|------|--------|------|
| A (텍스트만) | v2 메시지 그대로 | 없음 |
| B (스크린샷) | v2 메시지 + "실제 검색 결과입니다" | ChatGPT 스크린샷 |

### 배정
- 타겟이 적으므로 A/B 분리보다 **순차 테스트**
- 먼저 3곳에 B(스크린샷 포함)로 발송
- 응답률이 낮으면 다음 5곳에서 A vs B 비교

### 사전 준비
1. ChatGPT에서 "강남 레이저 제모 추천" 검색 → 스크린샷 촬영
2. "압구정 피부과 제모" 검색 → 스크린샷 촬영
3. 각 스크린샷에 해당 클리닉이 안 나온다는 게 보여야 함
4. 경쟁사(모리스의원)가 나오는 스크린샷도 준비 (비교용)

### 성공 기준
- **Pass**: 스크린샷 포함 시 3곳 중 1곳 이상 "관심" 이상 반응
- **Fail**: 3곳 모두 무응답 or 거절

### 측정할 것
| 지표 | 측정 방법 |
|------|----------|
| 응답률 | 답변 수 / 발송 수 |
| 응답 속도 | 발송~첫 답변 시간 |
| 반응 온도 | 거절(-1) / 무응답(0) / 질문(+1) / 관심(+2) / 미팅요청(+3) |
| "AI 검색" 언급 | 답변에서 AI 검색을 문제로 인식하는 표현 유무 |
| 추가 질문 내용 | 가격? 효과? 방법? → H-PRC, H-EFF 가설 힌트 |
