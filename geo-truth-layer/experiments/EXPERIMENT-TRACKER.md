# GEO Truth Layer — 실험 트래커

> 메타 가설: "AI가 병원을 정확히 찾고 설명하게 만드는 공식 정보 운영 서비스로 월 100~200만원 수익을 만들 수 있다"
> 원칙: 모든 행동은 가설이다. 측정 없는 행동은 낭비다.

---

## 가설 트리

메타 가설이 참이려면 아래 7개가 모두 참이어야 한다.
각 가설은 독립적으로 검증하고, 하나라도 거짓이면 피벗한다.

```
META: 클리닉 공식 정보 운영 서비스 = 월 100~200만
│
├── H-MKT  시장:   사장님이 AI 공식 정보 정확성 문제를 인식한다
├── H-RCH  도달:   카카오톡/전화로 의사결정자에게 닿는다
├── H-MSG  메시지: 공식 정보 운영+스크린샷이 관심을 만든다
├── H-PRC  가격:   셋업 30~50만 + 월 15~20만이 수용된다
├── H-DEL  전달:   1클리닉 셋업을 3시간 내에 완료 가능하다
├── H-EFF  효과:   셋업 후 AI 검색 노출이 실제 개선된다
└── H-RET  유지:   월간 리포트가 월정액 유지 이유가 된다
```

### 검증 순서 (의존성 기반)

```
Week 1-2:  H-RCH → H-MKT → H-MSG  (도달 → 문제 인식 → 메시지)
Week 2-3:  H-PRC                    (가격 수용)
Week 3-4:  H-DEL → H-EFF           (전달 → 효과)
Month 2+:  H-RET                    (유지)
```

도달이 안 되면 메시지 테스트를 못 하고, 문제 인식이 없으면 가격 테스트가 무의미하다.
따라서 앞쪽 가설부터 빠르게 kill-or-confirm 한다.

## 피벗 트리거

가설이 거짓으로 판명될 때의 대응:

| 가설 | 거짓 신호 | 피벗 옵션 |
|------|----------|----------|
| H-RCH | 10곳 DM 중 응답 0 | 채널 전환 (방문 영업, 네이버 카페, 협회) |
| H-MKT | "AI가 병원을 헷갈리는 문제? 관심 없어요" 반복 | 문제 리프레이밍 or 버티컬 전환 |
| H-MSG | 응답은 오는데 "그래서 뭐?" | 메시지 피벗 (공식 정보 오류 데모, 경쟁사 강조) |
| H-PRC | "비싸요" 반복 | 가격 인하 or 성과보수 모델 테스트 |
| H-DEL | 건당 6시간+ | 자동화 우선 or 서비스 범위 축소 |
| H-EFF | 4주 후 변화 없음 | 기술 재검토 or "인덱싱 증거"로 피벗 |
| H-RET | 첫 달 후 해지 | 리포트 형태 변경 or 가치 재설계 |
