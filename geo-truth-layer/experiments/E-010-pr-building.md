- 설계일: 2026-04-05
- 실행 시작: 2026-04-06 (일)

---

## 가설

> "AI로 사업자 돕는 방법을 실험 중"이라는 페르소나로 숏폼 콘텐츠를 매일 올리면,
> 30일 안에 재방문 시청자가 생기고, 90일 안에 PR이 세일즈 리드를 만든다.

---

## 하위 가설 트리

```
H-PR: PR 빌딩이 세일즈와 장기 자산을 만든다
│
├── H-PR-1  페르소나: "AI로 사업자 돕기 실험" 이 공감을 얻는다
├── H-PR-2  템플릿: 반복 포맷이 재방문을 만든다
├── H-PR-3  플랫폼: 유튜브 쇼츠가 한국 사업자에게 도달한다
├── H-PR-4  밈화: "사장님은 모릅니다"가 따라하기/공유를 유발한다
├── H-PR-5  전환: PR 시청자가 DM/문의로 전환된다
└── H-PR-6  지속: 실험 주제가 바뀌어도 팬이 유지된다
```

### 검증 순서

```
Week 1:    H-PR-1 → H-PR-3  (페르소나 공감 + 플랫폼 도달)
Week 2:    H-PR-2 → H-PR-4  (템플릿 재방문 + 밈화)
Week 3-4:  H-PR-5            (DM/문의 전환)
Month 2+:  H-PR-6            (주제 전환 후 팬 유지)
```

## 세일즈 실험과의 연동

### 세일즈 → PR 소재
```
E-001 결과 (카카오톡 도달) → Day 8~9 영상 소재
E-002 결과 (스크린샷 반응) → Day 10~11 영상 소재
E-003 결과 (셋업 시간)     → Day 16~17 영상 소재
```

### PR → 세일즈 리드
```
영상 시청자 DM → 무료 감사 제공 → 유료 전환
댓글 "우리 가게도 봐줘" → DM → 감사 → 전환
```

### 공유 데이터
- 세일즈 실험의 사장님 반응 → PR 콘텐츠 (허가 받고)
- PR 영상 반응 데이터 → 세일즈 메시지 톤 조정 참고
