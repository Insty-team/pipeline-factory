## 가설

> 카카오톡 채널/DM으로 중소 클리닉 의사결정자(원장 or 실장)에게 도달할 수 있다.

---

## 실험 설계

### 대상
| 클리닉 | 채널 | 연락처 상태 |
|--------|------|------------|
| 메티의원 | 카카오톡 채널 | 미확보 → 홈페이지에서 확인 |
| 유스타일나인 | 전화 02-553-2011 → 카카오톡 | 전화번호 있음 |
| 세진피부과 | 카카오톡 채널 | 미확보 → 홈페이지에서 확인 |

### 실행 순서
1. 각 클리닉 홈페이지 접속 → 카카오톡 ID / 전화번호 확보
2. 카카오톡 채널 검색 → 채널 있으면 DM
3. 채널 없으면 전화 → "마케팅 담당자분 계신가요?" → 카카오톡 ID 요청
4. DM 발송 (geo-truth-layer/sales/geo-target-list-and-outreach.md의 v2 메시지)

### 성공 기준
- **Pass**: 3곳 중 2곳 이상 DM 전달 성공 (읽음 확인 or 응답)
- **Fail**: 3곳 중 0~1곳만 도달

### 측정할 것
| 지표 | 측정 방법 |
|------|----------|
| 연락처 확보 성공/실패 | 각 클리닉별 기록 |
| DM 전달 성공/실패 | 카카오톡 읽음 표시 |
| 응답 유무 | 48시간 내 답변 |
| 응답 내용 분류 | 거절 / 관심 / 질문 / URL전달 |
| 의사결정자 도달 여부 | 답변자가 누구인지 |
| 소요 시간 | 연락처 확보~DM 발송까지 |
