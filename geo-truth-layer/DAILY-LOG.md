# 매일/매주 진척 트래커

> 목적: 단기 200~300만원 + 장기 Zero to One 목표를 향해 *오늘* 한 발짝 나갔는지 확인.
> 사용법: 하루 30초 — 일지 한 줄. 일요일 5분 — 주간 funnel 5수치. 월 1회 10분 — 분기 체크리스트 1/3.
> 진실 소스: 전략은 `STRATEGY.md`, 현재 상태는 `../HANDOFF.md`, 가설 학습은 `experiments/EXPERIMENT-TRACKER.md`. 이 파일은 *시간 축* 추적.

---

## 사용 규칙 (먼저 읽기)

### 1. 일일 한 줄 (Daily one-liner) — 매일 자기 전 30초

기록 방식: 아래 표에 새 행 1개. 그 이상 안 적어도 됨.

- **행동**: 오늘 무엇을 했는지 *동사*로 한 줄 (예: "앤드성형 카톡 채널 확인", "유스타일나인 팔로업 발송")
- **발송 N**: 오늘 신규 발송한 클리닉 수
- **응답 N**: 오늘 새로 받은 응답 수 (관심·거절·추가질문 모두 포함)
- **Phase**: 현재 STRATEGY.md 단계 (P0 / P1 / P2 / P3)

규칙:
- 0건이어도 적는다. 0이 며칠 연속이면 그것이 신호.
- "준비/리서치"만 한 날도 적는다 — 단, 발송·응답 수치엔 안 더해진다.
- 한 줄 채울 시간조차 없으면 다음 날 어제 칸을 채운다 (이틀 이상 밀리지 않기).

### 2. 주간 funnel snapshot — 일요일 5분

매주 일요일에 5수치 기록. 누적치는 자동으로 늘지 않으니 직접 더한다.

| 지표 | 정의 | 의미 |
|---|---|---|
| 발송 누적 | 프로젝트 시작 이후 총 발송 클리닉 수 (Wave 0 + Wave 1 + Wave 2...) | 발송량 게임 진척 |
| 응답 누적 | 답장 받은 클리닉 수 | 응답률 = 응답/발송 |
| 파일럿 active | 현재 무료 파일럿 진행 중 클리닉 수 | P0→P1 신호 |
| 유료 active | 유료 Setup 또는 Monthly 진행 중 클리닉 수 | P1→P2 신호 |
| MRR | 이번 달 반복 수익 (₩) | 200~300만 도달 직접 지표 |

### 3. 월간 mini-checkpoint — 월 1회 10분

매월 1일에 STRATEGY.md `4.1 분기 체크리스트` 7항목을 자체 점검. 분기(3개월)마다 풀 점검.

### 4. 변곡점 알람 — 자동 트리거

아래 중 하나라도 발동하면 *그날* STRATEGY.md를 다시 펼친다:
- 케이스 스터디 1건 확보 (P0→P1 진입)
- 50곳 누적 감사 + 패턴 수렴 점검
- Monthly 누적 10건 돌파
- 6주 누적 응답률 < 5%
- 강남언니/네이버Cue/Kakao 측 GEO 기능 출시 뉴스
