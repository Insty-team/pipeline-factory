## 치명적 문제 5가지

1. **구조화 데이터 완전 부재** — AI가 "이건 클리닉, 이건 가격, 이건 시술"을 구분 못 함
2. **JS 렌더링 의존** — AI 크롤러는 JS 실행 안 함 → 핵심 콘텐츠 못 읽을 가능성
3. **sitemap 5년 방치** — 가격 페이지가 sitemap에 없어 크롤링 우선순위 떨어짐
4. **메타 태그 없음** — AI가 페이지 요약할 근거가 없음
5. **llms.txt 없음** — AI에게 "이 사이트는 이런 곳" 안내 부재

## 잘 되어 있는 점

1. robots.txt 열려 있음 — 크롤러 차단 안 함
2. 가격이 이미지가 아닌 텍스트 — 구조화만 하면 즉시 읽힘
3. 60+ 시술 정보 풍부 — 소스 자체는 훌륭
4. 연락처/주소 명확 — LocalBusiness 스키마 바로 적용 가능
5. 다국어 지원 — 외국인 환자 AI 검색에도 대응 가능
