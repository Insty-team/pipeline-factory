## 치명적 문제

1. **구조화 데이터 없음** — sitemap은 좋은데 페이지 자체에 스키마가 없음
2. **Next.js 렌더링** — 비즈니스 정보가 JS 스트림에 매립, 크롤러가 못 읽음
3. **meta description 없음** — AI가 페이지 요약할 근거 부족
4. **AI-citability 콘텐츠 없음** — 비교 쿼리에 인용될 팩트 블록 부재
