## Phase 5: 2~3개월차 — 자동화 시작 (노동=수익 깨기)

### 자동화 우선순위

| 작업 | 현재 | 자동화 후 | 도구 |
|------|------|----------|------|
| GEO 감사 | 수동 30분 | 자동 5분 | geo-seo-claude 커스텀 |
| 리포트 생성 | 수동 1시간 | 자동 10분 | 템플릿 + 스크립트 |
| 주간 AI 검색 테스트 | 수동 30분/건 | 크론 자동 | 프롬프트 셋 + API |
| 월간 리포트 | 수동 2시간/건 | 자동 생성 | 데이터 파이프라인 |
| 크롤러 모니터링 | 수동 | 자동 알림 | cron + 스크립트 |

### 자동화 완료 후 시간 구조
```
관리 10건 기준:
  자동화 전: 10건 × 4시간/월 = 40시간/월 (주말만으론 불가능)
  자동화 후: 10건 × 0.5시간/월 = 5시간/월 + 영업 4시간 = 9시간/월

→ 관리 10건 × 15만원 = 150만원/월, 투입 9시간
→ 시급 환산: ~16,700원/시간 → 시스템이 쌓일수록 올라감
```

### 빌드할 자동화 시스템
1. **GEO 감사 자동화** — geo-seo-claude 포크, 뷰티 특화 템플릿, 한국어 리포트 출력
2. **모니터링 대시보드** — Supabase + Cloudflare Pages (기존 인프라 활용)
3. **AI 검색 테스트 크론** — 고정 프롬프트 셋을 주간 자동 실행, 결과 저장
4. **월간 리포트 자동 생성** — 데이터 → 템플릿 → PDF
